# 错题本分析助手

> 拍照上传错题，帮你分析为什么做错，并出几道类似题巩固！

---

## 🎯 这是什么

一个帮学生（和家长）分析错题、找出原因、巩固练习的Skill。

---

## 🚀 使用方法

```
"帮我分析这道题"
"上传错题"
"这道数学题怎么讲"
"为什么做错了"
"出一道类似的题"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 题目 | 错题内容 | 文字/拍照 |
| 正确答案 | 参考答案 | 14 |
| 你的答案 | 你写的 | 13 |
| 学科/年级 | 什么学科 | 小学三年级数学 |

---

## 📦 输出内容

1. **错误原因分析** - 为什么会错
2. **知识点讲解** - 相关知识点科普
3. **正确思路** - 一步步教你
4. **同类练习题** - 巩固训练
5. **学习建议** - 避免下次犯错

---

## 💡 功能特点

- 📷 支持拍照上传
- 🔍 精准定位错误原因
- 📚 知识点系统讲解
- 📝 举一反三练习题
- 👨‍👩‍👧 家长辅导指南

---

## ⚡ 适用学科

| 学科 | 支持内容 |
|-----|---------|
| 语文 | 拼音、字词、阅读、作文 |
| 数学 | 计算、应用题、几何 |
| 英语 | 语法、阅读、完形、作文 |
| 物理 | 概念、公式、计算、分析 |
| 化学 | 方程式、反应、计算 |

---

## 👨‍👩‍👧 家长辅导建议

- 先让孩子说说自己怎么想的
- 用孩子能懂的方式讲
- 讲完出类似题检验
- 几天后要复习
