## 📖 如何使用 Skills

### 方式一：直接对话触发（推荐）

在WorkBuddy中直接描述你的需求，例如：

```
# 触发 anti-pua
"老板说'你怎么连这点事都做不好'，怎么怼回去？"
"领导让我周末加班，我说有事，他说大家都在拼，怎么回？"
```

WorkBuddy会自动识别并加载对应的Skill。

### 方式二：手动选择Skill

从左侧边栏的「专家」入口，进入「专家中心」，浏览分类后选择对应Skill开始对话。

## 🛠️ 如何创建新的 Skill

### Step 1：规划Skill

问自己几个问题：
- 这个Skill解决什么问题？
- 用户会怎么说触发这个Skill？
- 需要哪些参考资料或脚本？

### Step 2：创建目录结构

```
skills-ai/
└── your-skill-name/              # Skill文件夹（英文、中划线）
    ├── SKILL.md                  # ⭐ 必须：Skill配置
    └── references/               # 可选：参考资料
        └── xxx.md
    └── scripts/                  # 可选：自动化脚本
        └── xxx.py
    └── assets/                   # 可选：静态资源
        └── xxx.png
```

### Step 3：编写 SKILL.md

SKILL.md 是Skill的核心，包含两部分：

```markdown
---
name: "Skill名称"
description: "这个Skill做什么的，什么时候触发它"
---

# 标题

你的详细说明...
```

**示例：**

```markdown
---
name: "小红书文案助手"
description: "帮助用户创作小红书风格的文案，包括标题、emoji搭配、正文结构。适用于需要发布小红书笔记、写种草文案、想标题等场景。"
---

# 小红书文案助手

## 功能
- 生成吸引眼球的标题
- 优化emoji搭配
- 撰写种草文案
- 提供hashtag建议

## 格式要求
- 标题：emoji + 痛点/好奇点
- 正文：开头钩子 + 干货/体验 + 结尾引导
- 结尾：行动号召 + 评论区互动
```

### Step 4：安装Skill

复制你的Skill文件夹到WorkBuddy的skills目录：

```
# 路径
C:\Users\<你的用户名>\.workbuddy\skills\

# 或者在文件管理器中：
# 打开 %USERPROFILE%\.workbuddy\skills\
# 将你的skill文件夹粘贴进去
```

### Step 5：重启WorkBuddy

重启后，新Skill会自动加载，可以搜索到了。

## 📝 SKILL.md 编写规范

### 必需字段

| 字段 | 说明 | 示例 |
|-----|------|------|
| `name` | Skill名称（英文、中划线） | `"职场邮件助手"` |
| `description` | 描述：做什么+何时触发+核心原则 | 见下方示例 |

### description 编写技巧

description 决定WorkBuddy何时触发你的Skill，要包含：
1. **功能描述**：这个Skill做什么
2. **触发场景**：什么情况下应该用它
3. **核心特点**：有什么独特价值

**好例子：**
```
"帮助用户撰写专业的职场邮件，包括邮件模板、措辞优化、格式规范。适用于写辞职信、商务沟通、投诉邮件、申请邮件等场景。"
```

**差例子：**
```
"帮助写邮件"  # 太模糊
```

### 正文编写建议

- 使用**命令式语气**（"生成"、"提供"、"按照"），而非"你可以"
- 分层结构，用标题组织内容
- 包含具体示例，让AI理解输出风格
- 保持简洁，详细信息放 `references/` 目录

## ❓ 常见问题

**Q: Skill放在哪里？**
A: `C:\Users\<用户名>\.workbuddy\skills\`

**Q: 一个目录可以放多个Skill吗？**
A: 不可以，每个Skill需要一个独立文件夹。

**Q: Skill不生效怎么办？**
A: 1) 确认SKILL.md格式正确 2) 重启WorkBuddy 3) 检查description是否包含触发关键词

**Q: 可以分享给其他人吗？**
A: 可以，把Skill文件夹压缩分享，对方放到自己的skills目录即可。
