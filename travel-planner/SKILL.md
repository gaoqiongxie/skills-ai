## 七大核心能力

### 一、行程规划

按天输出行程，格式如下：

```
Day 1 · 【主题】
上午：A景点（建议游览2小时）→ 步行5分钟 → B景点
午餐：xxx餐厅（人均50，推荐xxx菜）
下午：C景点 → 傍晚 D打卡点（绝佳日落时间：17:30-18:30）
晚餐：xxx街区觅食
住宿：xxx区域（离次日行程近）

💡 贴士：xxx月份人多，建议提前2周订票
⚠️ 避坑：D景点淡季不开放，出发前确认
```

**行程原则**：
- 同区域的景点安排同一天，减少来回奔波
- 爬山/高强度活动安排在体力好的上午
- 留一个下午"休闲备用"，不要把每天排满
- 带娃/老人行程强度减半
- 自驾注意每日驾驶时长 ≤5小时

### 二、住宿推荐

```
推荐住在：xxx区域
理由：距离主要景点近 / 交通方便 / 美食集中

住宿类型建议：
- 🏨 连锁酒店（如汉庭/全季）：稳定，适合家庭
- 🏡 民宿（如途家/小猪）：有特色，适合情侣/文艺范
- 🛕 特色住宿（如古镇民宿/露营）：体验独特，提前2-4周订

参考价格区间：
- 经济型：100-200元/晚
- 中档：200-500元/晚
- 品质/特色：500元以上
```

**选址原则**：离次日第一个景点近，不要为了便宜住太远。

## 热门目的地速查

### 华东
- **上海**：城市游，购物/美食/迪士尼/江南水乡周边
- **杭州**：西湖/千岛湖/乌镇，适合2-3天
- **苏州**：园林/古镇，适合2天
- **黄山**：2天1夜+，需要一定体力

### 西南
- **云南**：大理（2天）+丽江（2天）+香格里拉（2天），推荐7-10天连游
- **西藏**：至少7天，需提前办边防证，高反严重者慎去
- **四川**：成都（美食3天）+九寨沟/峨眉山/稻城亚丁（5天+）
- **贵州**：贵阳+黔东南（苗寨），小众但值得

### 华南
- **广州/深圳**：城市游，美食/购物/主题乐园
- **厦门**：鼓浪屿/土楼，3天合适
- **桂林**：山水，阳朔，3-4天

### 华北
- **北京**：长城/故宫/颐和园，3-5天
- **山西**：平遥/大同，历史控必去，2-3天

### 西北
- **新疆**：伊犁+喀纳斯+南疆，建议10天+（地域广，飞机转场）
- **甘肃**：敦煌+张掖彩虹山+黄河石林，自驾5-7天
- **青海**：青海湖+茶卡盐湖，3-4天

### 海岛
- **三亚**：度假型，3-5天，冬季最佳
- **海南全岛**：自驾环岛，5-7天
- **舟山/嵊泗**：上海周边出海，周末2天

## 节假日出行提醒

| 假期 | 建议策略 |
|------|---------|
| 五一/十一黄金周 | 提前1-2个月订票/酒店；避开北京故宫/张家界/九寨沟等顶流 |
| 春节 | 热门景区爆满；南方暖冬目的地抢手；部分餐厅营业不稳定 |
| 清明/端午 | 短途+周边游为主；爬山类景区人多 |
| 暑假（7-8月） | 亲子游高峰；北方草原/海岛最热；西藏是好时机 |
| 元旦小长假 | 南下避寒（三亚/云南/广州）是主流 |

## 特殊场景规划

### 带娃游（有小孩）
- 优先选：设施完善的景区（有厕所/餐厅/休息区）
- 避开：长途爬山、偏远无补给路段
- 住宿：选有儿童设施的连锁酒店
- 行程：每天不超过2-3个景点，留充足休息时间

### 自驾游
- 出发前：车辆保养检查/保险确认/导航下载离线地图（山区信号差）
- 路线：优先走国道/省道，风景更好
- 住宿：提前订好，不要边开边找
- 紧急联系：高速救援电话 12122

### 穷游/低预算
- 住：青年旅舍/民宿多人间（50-100元/晚）
- 吃：早市/菜市场/小街巷（避开旅游区价格）
- 玩：很多城市公园/老街免费，合理搭配
- 行：高铁二等座/大巴，提前购票

## 注意事项

1. **证件**：身份证必带；部分景区（新疆/西藏部分区域）需办特别证件，提前了解
2. **手机信号**：偏远景区无信号，提前下载离线地图（高德/奥维）
3. **预约制度**：热门景区（故宫/黄山/九寨沟）需提前预约，注意名额
4. **高反**：海拔超过3000米需注意，西藏/香格里拉/稻城亚丁尤其注意
5. **购物**：景区伴手礼普遍贵，机场/高铁站价格更实惠，网上更划算
6. **拍照**：部分寺庙/博物馆禁止拍照，进门看提示
