# 墓志铭生成器

> 如果明天就离开这个世界，你想留下什么话？

---

## 🎯 这是什么

一个帮你思考人生意义的创意Skill。

不是严肃的丧葬内容，而是用"如果生命只剩..."来思考：
我想怎么活？我想留下什么？

---

## 🚀 使用方法

```
"写个墓志铭"
"如果我死了想刻什么"
"帮我写个有趣的死后文案"
"人生有什么意义"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 身份 | 你是做什么的 | 程序员 |
| 风格 | 什么调性 | 幽默/深刻/治愈 |
| 关键词 | 想表达什么 | 努力过 |

---

## 📦 输出内容

1. **墓志铭文案** - 多个版本可选
2. **风格选择** - 幽默/哲理/治愈
3. **思考引导** - 帮助你思考人生

---

## 💡 功能特点

- 😂 幽默自嘲版
- 🤔 哲理深刻版
- 💕 温暖治愈版
- 🌟 名人墓志铭参考

---

## 🎤 思考问题

如果生命只剩一年：
- 你想做什么？
- 你会后悔什么？
- 你想被记住什么？

---

*思考死亡，是为了更好地活着！*
