# 废话翻译官

> **来源**: 原创 - 职场/社交沟通技巧
>
> **灵感来源**: 网络废话文学、职场黑话、体制内话术

## 功能

将大白话翻译成"听不懂"版本，应对听不懂人话的人。

## 使用方法

```
用户: "把'你的方案太烂了'说得高大上"
助手: 自动翻译

用户: "领导说的'战略性调整'是什么意思？"
助手: 翻译成人话
```

## 核心能力

1. **大白话 → 高大上**: 把直话翻译成听不懂的版本
2. **专业术语包装器**: 用术语替换简单词汇
3. **废话文学生成器**: 制造听起来很有道理但什么都没说的内容
4. **委婉拒绝话术**: 不想帮忙时的官方话术
5. **听不懂预警**: 识别真正的意思

## 示例

| 场景 | 大白话 | 翻译后 |
|-----|-------|--------|
| 批评 | 你的方案太烂了 | 在底层逻辑和顶层设计层面还有很大的优化空间 |
| 拒绝 | 我不想帮你 | 从全局视角来看，这可能不是最高优先级的资源配置 |
| 听不懂 | 听不懂 | 这个信息在我的认知边界之外 |
