# 菜谱随机抽

> 不知道做什么菜？告诉我有什么食材，帮你想菜谱！

---

## 🎯 这是什么

一个帮你决定"今天做什么菜"的Skill。

输入食材，给你推荐；或者随机抽一道菜给你！

---

## 🚀 使用方法

```
"随机推荐一道菜"
"冰箱里有鸡蛋和西红柿，能做什么？"
"来一道简单的快手菜"
"教我做红烧肉"
"10分钟能做什么菜？"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 食材 | 冰箱里有什么 | 鸡肉、土豆、西红柿 |
| 难度 | 想做简单的还是复杂的 | 入门级、快手菜 |
| 时间 | 有多少时间 | 10分钟、半小时 |
| 口味 | 想吃什么口味 | 辣的、清淡的 |
| 人数 | 做给几个人 | 1人、2人 |

---

## 📦 输出内容

1. **推荐菜名** - 根据食材推荐
2. **难度/时间** - 预估
3. **所需食材** - 主料+辅料+调料
4. **详细步骤** - 一步步教你做
5. **小技巧** - 让菜更好吃的秘诀

---

## 💡 功能特点

- 🧂 食材匹配 → 自动推荐可做菜谱
- 🎲 随机抽取 → 每天不一样的惊喜
- ⏱️ 时间筛选 → 找到适合你的快手菜
- 📊 难度分级 → 从入门到大师
