# 电梯演讲生成器

> 怎么在30秒内说清楚一件事？告诉我场景，帮你生成！

---

## 🎯 这是什么

一个帮你生成简洁有力的"电梯演讲"的Skill。

30秒、1句话，让对方记住你想说的！

---

## 🚀 使用方法

```
"帮我写个30秒自我介绍"
"怎么向领导汇报项目进展？"
"面试怎么介绍自己？"
"生成一个产品介绍"
"帮我写个电梯演讲"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 介绍对象 | 人/项目/产品/公司 | Java开发 |
| 场景 | 面试/汇报/社交/路演 | 面试 |
| 时长 | 1句话/30秒/1分钟 | 30秒 |
| 关键信息 | 你的经历/项目亮点 | 3年经验 |

---

## 📦 输出内容

1. **演讲稿** - 符合时长的精炼版本
2. **不同版本** - 30秒/1分钟可选
3. **示例参考** - 看看别人怎么说

---

## 💡 功能特点

- 🎯 精准提炼核心信息
- ⏱️ 严格控制时长
- 👔 看人下菜碟（专业/领导/用户）
- 📝 提供多种场景模板

---
