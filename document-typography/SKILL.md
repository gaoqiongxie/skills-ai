# 📐 Document Typography — 文档排版质量控制

让 AI 生成的文档在排版上达到出版级标准。

---

## 🎯 核心能力

| 能力 | 说明 |
|------|------|
| **孤行寡段检测** | 自动识别段末孤行、段首寡行 |
| **编号对齐检查** | 多级列表编号连续性、缩进一致性 |
| **字体层级审查** | 标题层级跳跃、字体混用、字号不连续 |
| **间距规范** | 段前段后、行距、页边距统一 |
| **标点规范** | 中英文标点混用、全半角统一 |
| **图表排版** | 图注表注位置、编号连续性 |

---

## 🔍 排版检查清单

### 文本排版

| 检查项 | 问题示例 | 修复建议 |
|--------|---------|---------|
| **孤行 (Widow)** | 段落最后一行单独出现在页首 | 调整段落分页或改写 |
| **寡行 (Orphan)** | 段落第一行单独留在页尾 | 同上 |
| **标题孤行** | 标题后仅跟一行内容即分页 | 强制标题与下文同页 |
| **行距不一致** | 部分段落 1.5 倍，部分 1.0 倍 | 统一为 1.5 倍行距 |
| **段前段后混乱** | 有的 6pt 有的 12pt | 统一模板 |

### 编号与列表

| 检查项 | 问题示例 | 修复建议 |
|--------|---------|---------|
| **编号断层** | 1. 2. 4. （缺少 3） | 补全或重新编号 |
| **层级跳跃** | 1. → 1.1.1. （跳过 1.1） | 补全中间层级 |
| **缩进不一致** | 同级列表缩进量不同 | 统一制表位 |
| **混合编号** | 同一列表混用 1. 和 (1) | 统一风格 |

### 字体与样式

| 检查项 | 问题示例 | 修复建议 |
|--------|---------|---------|
| **字体混用** | 正文混用宋体和微软雅黑 | 统一正文字体 |
| **字号跳跃** | 标题层级：24pt → 18pt → 20pt | 递减：24→20→16→14 |
| **颜色过多** | 正文出现 5 种以上颜色 | 限制为 2-3 种主色 |
| **加粗滥用** | 大段文字加粗 | 仅关键词加粗 |
