## 收纳原则

### 核心原则（怦然心动整理法）

```
📋 【怦然心动整理法】

1️⃣ 集中：把所有同类物品放一起
   例：把所有衣服放床上

2️⃣ 挑选：留下让你心动的
   扔掉/捐掉/卖掉不心动的

3️⃣ 分类：
   - 当季穿 / 过季
   - 上装 / 下装 / 内衣 / 配件
   - 纪念品 / 实用

4️⃣ 固定位置：每类东西有固定的家

5️⃣ 归位：用完放回原位
```

### 收纳黄金法则

```
✓ 站起来，别躺着
  → 竖着放，一眼能看到

✓ 化零为整
  → 用收纳盒统一

✓ 动线最短
  → 常用物品放在方便拿的地方

✓ 二八原则
  → 80%藏起来，20%展示
```

## 衣柜收纳

### 1. 空间规划

```
┌─────────────────────────────────┐
│            👔 衣柜              │
├─────────────────────────────────┤
│  上层区（叠放区）              │
│  - 被子/过季被褥              │
│  - 行李箱                     │
│  - 收纳盒（不常用物品）        │
├─────────────────────────────────┤
│  中层区（悬挂区）              │
│  - 长衣区（连衣裙/风衣）       │
│  - 短衣区（上衣/外套）         │
│  - 裤子区                      │
├─────────────────────────────────┤
│  下层区（抽屉+叠放）           │
│  - 抽屉：内衣/袜子/配饰        │
│  - 叠放：毛衣/T恤              │
└─────────────────────────────────┘
```

### 2. 衣服折叠法

**竖着折叠法（推荐）**：

```
每件衣服：
1. 袖子往里折
2. 从领口往下折
3. 再对折
4. 竖着放进收纳盒

效果：
- 竖着放，一眼看到所有衣服
- 抽一件不会乱
```

**特殊衣物**：

| 衣物 | 收纳方式 |
|-----|---------|
| 毛衣 | 折叠竖放（不挂，会变形） |
| 牛仔裤 | 对折后叠放/卷起来 |
| 连衣裙 | 挂起来 |
| 衬衫 | 挂起来（可以叠放但会皱） |
| 内衣 | 叠放抽屉/收纳盒 |
| 袜子 | 对折后叠放（不滚成一团） |
| 丝巾 | 卷起来/折叠挂起 |

### 3. 收纳好物

```
推荐物品：

✅ 抽屉收纳格
   - 内衣袜子分类
   - 价格：¥20-50

✅ 植绒衣架
   - 防滑、节省空间
   - 价格：¥5-15/个

✅ 收纳箱/收纳盒
   - 过季衣物/被子
   - 价格：¥30-100

✅ 伸缩杆
   - 增加挂衣空间
   - 价格：¥20-50

✅ 真空收纳袋
   - 被子/厚衣服压缩
   - 价格：¥20-50/套
```

## 厨房收纳

### 1. 空间规划

```
┌─────────────────────────────────┐
│            🍳 厨房              │
├─────────────────────────────────┤
│  上层（轻/不常用）              │
│  - 备用调料                     │
│  - 偶尔用的锅具                 │
├─────────────────────────────────┤
│  中层（常用，伸手可及）          │
│  - 每天用的调料                 │
│  - 常用锅具                     │
│  - 碗碟                         │
├─────────────────────────────────┤
│  下层（重/常用）                │
│  - 米面                         │
│  - 油                           │
│  - 大桶饮料                     │
├─────────────────────────────────┤
│  冰箱                           │
│  - 门上：调味品/饮料             │
│  - 上层：剩菜/乳制品            │
│  - 下层：生肉/蔬菜              │
│  - 抽屉：水果/蔬菜              │
└─────────────────────────────────┘
```

### 2. 收纳技巧

**调料区**：
```
- 常用调料放台面/吊柜
- 不常用调料放柜子
- 用统一的调料瓶（更整洁）
- 贴标签（盐/糖/鸡精分清）
```

**冰箱**：
```
- 生熟分开
- 密封保存
- 定期清理（过期扔掉）
- 蔬菜/水果分开放
- 用保鲜盒/保鲜袋
```

**锅具**：
```
- 锅盖倒扣放
- 竖着放（节省空间）
- 不锈钢盆套起来
- 用收纳架
```

### 3. 收纳好物

```
推荐物品：

✅ 旋转调料盘
   - 方便拿取
   - 价格：¥30-80

✅ 保鲜盒套装
   - 冰箱收纳
   - 价格：¥50-150

✅ 挂墙置物架
   - 勺子/锅铲挂起来
   - 价格：¥30-100

✅ 水槽沥水架
   - 碗碟沥水
   - 价格：¥50-150

✅ 米桶
   - 密封防潮
   - 价格：¥30-100

```

| 抽屉 | 收纳内容 | 收纳方式 |
|-----|---------|---------|

**AI输出**：
