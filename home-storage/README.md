# 家居收纳指南

> 家里乱糟糟？告诉我哪个空间，我来帮你收纳！

---

## 🎯 这是什么

一个帮你整理家居的Skill。

衣柜/厨房/卫生间/客厅...
各种空间的收纳方案！

---

## 🚀 使用方法

```
"衣柜怎么收纳"
"厨房太乱了"
"卫生间收纳"
"帮我整理"
"收纳技巧"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 空间 | 哪里乱 | 衣柜 |
| 物品 | 什么东西 | 衣服太多 |
| 问题 | 具体问题 | 找不到衣服 |

---

## 📦 输出内容

1. **空间规划** - 怎么分区
2. **收纳技巧** - 具体方法
3. **收纳好物** - 推荐神器
4. **操作步骤** - 一步步来

---

## 💡 功能特点

- 👔 衣柜收纳
- 🍳 厨房收纳
- 🚿 卫生间收纳
- 🛋️ 客厅收纳
- 📦 抽屉收纳

---

## 🎤 收纳原则

| 原则 | 说明 |
|-----|------|
| 站起来 | 竖着放，一眼看到 |
| 化零为整 | 用收纳盒统一 |
| 动线最短 | 常用物品放方便处 |
| 二八原则 | 80%藏/20%展示 |
