## 🔧 多语言实现

### ☕ Java Spring Boot

```java
@Scheduled(cron = "0 0 3 * * ?")  // 每天3点
public void dailyTask() {}
```

### 🗄️ MySQL Event

```sql
CREATE EVENT daily_cleanup
ON SCHEDULE EVERY 1 DAY STARTS '2026-01-01 03:00:00'
DO CALL cleanup_procedure();
```

### 🐧 Linux Crontab

```bash
0 3 * * * /path/to/script.sh
```
