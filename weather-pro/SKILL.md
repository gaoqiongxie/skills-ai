## 🌐 数据源整合

### 中国气象局（CMA）
- **优势**: 国内权威，台风/暴雨预警准确
- **数据**: 实况、预报、预警、雷达、卫星云图
- **API**: https://data.cma.cn/

### OpenWeatherMap
- **优势**: 全球覆盖，历史数据丰富
- **数据**: 温度、湿度、气压、风速、紫外线、空气质量
- **API**: https://openweathermap.org/api

### 彩云天气
- **优势**: 分钟级降水预报，雷达外推准确
- **数据**: 降水预报、雷达图、空气质量
- **API**: https://caiyunapp.com/

## 📊 气象数据解读框架

### 基础要素解读

| 要素 | 正常范围 | 异常含义 | 影响 |
|------|---------|---------|------|
| **温度** | 因地而异 | 偏高/偏低 | 人体舒适度、能耗、农业 |
| **湿度** | 40%-70% | 过高/过低 | 体感、健康、物品保存 |
| **气压** | 1013hPa | 骤降/骤升 | 天气变化、人体反应 |
| **风速** | <10m/s | >17m/s大风 | 出行、建筑、航空 |
| **能见度** | >10km | <1km大雾 | 交通、航空、航海 |
| **降水量** | 因地而异 | 暴雨/干旱 | 洪涝、农业、供水 |

### 进阶指标

| 指标 | 含义 | 应用场景 |
|------|------|---------|
| **露点温度** | 水汽饱和温度 | 判断体感闷热程度 |
| **体感温度** | 综合温湿度风速 | 穿衣、户外活动 |
| **紫外线指数** | UV辐射强度 | 防晒、皮肤保护 |
| **空气质量指数(AQI)** | PM2.5/臭氧等 | 健康、运动建议 |
| **K指数** | 大气不稳定度 | 雷暴预报 |
| **CAPE** | 对流有效位能 | 强对流潜势 |

## 🎓 气象科普

### 常见气象现象解释

| 现象 | 成因 | 影响 | 趣味知识 |
|------|------|------|---------|
| **梅雨** | 准静止锋 | 长江中下游6-7月连绵阴雨 | 入梅出梅看副高 |
| **寒潮** | 西伯利亚冷空气南下 | 大范围剧烈降温 | 24小时降温8°C以上才算 |
| **台风** | 热带洋面暖湿气流旋转 | 强风暴雨 | 西北太平洋是全球台风最多海域 |
| **厄尔尼诺** | 东太平洋海温异常偏高 | 全球气候异常 | 名字是西班牙语"圣婴" |
| **拉尼娜** | 东太平洋海温异常偏低 | 与厄尔尼诺相反 | 名字是"圣女" |
| **雷暴** | 强对流发展 | 雷电大风冰雹 | 云中电荷分离导致放电 |
| **雾霾** | 污染物+静稳天气 | 健康影响 | PM2.5是指直径<2.5微米的颗粒物 |
| **沙尘暴** | 强风+干燥沙源 | 空气质量恶化 | 蒙古气旋是主要动力 |

### 气象谚语科学解读

```
"朝霞不出门，晚霞行千里"
→ 科学原理：朝霞说明东方有云层，水汽东移可能带来降水；
   晚霞说明西方放晴，天气趋于稳定

"燕子低飞要下雨"
→ 科学原理：降雨前气压降低、湿度增大，昆虫翅膀沾湿飞得低，
   燕子捕食随之低飞

"天上钩钩云，地上雨淋淋"
→ 科学原理：钩卷云（cirrus uncinus）是暖湿气流 approaching 的标志，
   通常预示未来24-48小时有降水系统移入
```

## 💡 使用示例

### 示例1：个人出行气象分析
```
用户：帮我分析一下明天北京的气象条件，适合去长城吗？

助手：→ 调取多源数据（CMA + 彩云 + OWM）
     → 分析：
       - 温度：8-18°C，体感偏凉
       - 风力：西北风4-5级，阵风6级
       - 能见度：>10km，良好
       - 降水：无
     → 建议：
       ✅ 适合出行，但需注意防风保暖
       ⚠️ 长城上风更大，建议带防风外套
       📸 能见度好，适合拍照
```

### 示例2：农业气象服务
```
用户：山东德州的小麦，最近气象条件怎么样？灌浆期要注意什么？

助手：→ 分析德州未来7天气象：
       - 温度：20-28°C，适宜灌浆
       - 降水：x日有小雨10mm，补充土壤水分
       - 日照：日均8小时，有利于光合作用
       - 风险：x日可能出现干热风（>30°C+相对湿度<30%+风速>3m/s）
     → 建议：
       ✅ 总体条件适宜
       ⚠️ x日干热风风险，提前灌溉或喷施叶面肥
```

### 示例3：航空气象分析
```
用户：明天上午上海浦东机场飞行条件怎么样？

助手：→ 分析PVG机场气象：

     → 建议：
       ⚠️ 早间航班可能有延误
       ✅ 09时后条件改善
```
