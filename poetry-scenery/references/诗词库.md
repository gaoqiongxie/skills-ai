# 诗词库 - 按风景分类

本库收录适合各种风景场景的经典诗词，涵盖山水、瀑布、江河、湖泊、大海、花草、雪景、月亮、日出日落等场景。

---

## 一、瀑布 🌊

### 经典之选

**《望庐山瀑布》李白·唐**
```
日照香炉生紫烟，遥看瀑布挂前川。
飞流直下三千尺，疑是银河落九天。
```
> 适用：庐山瀑布及任何壮阔瀑布场景

**《望庐山瀑布》其二 李白**
```
日夜望庐山瀑布，通天一门三千尺。
```
> 简短版，适合快速引用

**《瀑布》施肩吾·唐**
```
豁开青冥颠，泻出万丈泉。
如裁一条素，白日悬秋天。
```
> 适合拍照发朋友圈

**《湖口望庐山瀑布》张九龄·唐**
```
万丈红泉落，迢迢半紫氛。
奔流下杂树，洒落出重云。
```

### 小众之选

**《瀑布》章孝标·唐**
```
秋河溢长空，天洒万丈布。
深雷隐云壑，孤电明岩树。  
```

## 二、山水 🏔️

### 壮阔山河

**《望岳》杜甫·唐**
```
岱宗夫如何？齐鲁青未了。
造化钟神秀，阴阳割昏晓。
荡胸生层云，决眦入归鸟。
会当凌绝顶，一览众山小。
```
> 适合登山、特别是登泰山

**《题西林壁》苏轼·宋**
```
横看成岭侧成峰，远近高低各不同。
不识庐山真面目，只缘身在此山中。
```
> 适合庐山及任何云雾缭绕的山

**《登山》李涉·唐**
```
终日昏昏醉梦间，忽闻春尽强登山。
因过竹院逢僧话，偷得浮生半日闲。
```

### 山水意境

**《山居秋暝》王维·唐**
```
空山新雨后，天气晚来秋。
明月松间照，清泉石上流。
竹喧归浣女，莲动下渔舟。
随意春芳歇，王孙自可留。
```
> 适合山林、清泉、竹林场景

**《鸟鸣涧》王维·唐**
```
人闲桂花落，夜静春山空。
月出惊山鸟，时鸣春涧中。
```
> 适合静谧山谷、溪涧

**《山中》王维·唐**
```
荆溪白石出，天寒红叶稀。
山路元无雨，空翠湿人衣。
```

### 必背名句**

- "山不在高，有仙则名" —— 刘禹锡
- "山高月小，水落石出" —— 苏轼
- "青山绿水，白草红叶黄花" —— 白朴

## 三、江河湖海 🌊

### 长江黄河

**《念奴娇·赤壁怀古》苏轼·宋**
```
大江东去，浪淘尽，千古风流人物。
```
> 适合长江三峡、壮阔江面

**《登高》杜甫·唐**
```
无边落木萧萧下，不尽长江滚滚来。
```

**《凉州词》王之涣·唐**
```
黄河远上白云间，一片孤城万仞山。
羌笛何须怨杨柳，春风不度玉门关。
```

### 湖泊

**《饮湖上初晴后雨》苏轼·宋**
```
水光潋滟晴方好，山色空蒙雨亦奇。
欲把西湖比西子，淡妆浓抹总相宜。
```
> 西湖及任何美丽湖泊！

**《望洞庭》刘禹锡·唐**
```
湖光秋月两相和，潭面无风镜未磨。
遥望洞庭山水翠，白银盘里一青螺。
```

**《宿建德江》孟浩然·唐**
```
移舟泊烟渚，日暮客愁新。
野旷天低树，江清月近人。
```

### 大海

**《观沧海》曹操·东汉**
```
东临碣石，以观沧海。
水何澹澹，山岛竦峙。
树木丛生，百草丰茂。
秋风萧瑟，洪波涌起。
日月之行，若出其中；
星汉灿烂，若出其里。
```
> 适合看海、观潮

**《面朝大海，春暖花开》海子·现代**
```
从明天起，做一个幸福的人
喂马、劈柴，周游世界
从明天起，关心粮食和蔬菜
我有一所房子，面朝大海，春暖花开
```
> 现代诗，适合发朋友圈

**《致大海》普希金·俄**
```
再见吧，自由的元素！
```

## 四、花草 🌸

### 春天百花

**《江畔独步寻花》杜甫·唐**
```
黄四娘家花满蹊，千朵万朵压枝低。
留连戏蝶时时舞，自在娇莺恰恰啼。
```
> 适合油菜花田、花海、桃花

**《游园不值》叶绍翁·宋**
```
应怜屐齿印苍苔，小扣柴扉久不开。
春色满园关不住，一枝红杏出墙来。
```
> 适合杏花、春天花园

**《清明》杜牧·唐**
```
清明时节雨纷纷，路上行人欲断魂。
借问酒家何处有，牧童遥指杏花村。
```

### 荷花

**《晓出净慈寺送林子方》杨万里·宋**
```
毕竟西湖六月中，风光不与四时同。
接天莲叶无穷碧，映日荷花别样红。
```
> 适合西湖荷花及任何荷花场景

**《如梦令》李清照·宋**
```
常记溪亭日暮，沉醉不知归路。
兴尽晚回舟，误入藕花深处。
争渡，争渡，惊起一滩鸥鹭。
```

### 梅花

**《梅花》王安石·宋**
```
墙角数枝梅，凌寒独自开。
遥知不是雪，为有暗香来。
```

**《卜算子·咏梅》陆游·宋**
```
驿外断桥边，寂寞开无主。
已是黄昏独自愁，更著风和雨。
无意苦争春，一任群芳妒。
零落成泥碾作尘，只有香如故。
```

### 菊花

**《饮酒·其五》陶渊明·晋**
```
采菊东篱下，悠然见南山。
山气日夕佳，飞鸟相与还。
```
> 适合秋菊、田园

### 通用花草

**《诗经·蒹葭》**
```
蒹葭苍苍，白露为霜。
所谓伊人，在水一方。
```
> 适合芦苇、湿地

## 五、月亮 🌙

### 思乡月

**《静夜思》李白·唐**
```
床前明月光，疑是地上霜。
举头望明月，低头思故乡。
```
> 月亮场景万能之选！

**《月夜忆舍弟》杜甫·唐**
```
露从今夜白，月是故乡明。
```

### 哲理月

**《水调歌头》苏轼·宋**
```
人有悲欢离合，月有阴晴圆缺，
此事古难全。
但愿人长久，千里共婵娟。
```

**《春江花月夜》张若虚·唐**
```
春江潮水连海平，海上明月共潮生。
滟滟随波千万里，何处春江无月明。
```
> 月下江景最美！

### 边塞月

**《出塞》王昌龄·唐**
```
秦时明月汉时关，万里长征人未还。
但使龙城飞将在，不教胡马度阴山。
```

## 六、日出日落 🌅

### 日出

**《日出》**
```
日出江花红胜火，春来江水绿如蓝。
——白居易《忆江南》
```
> 适合日出时分的江面

**《望岳》杜甫·唐**
```
登鹳雀楼》王之涣·唐**
```
白日依山尽，黄河入海流。
欲穷千里目，更上一层楼。
```
> 适合登高望远、日落时分

### 夕阳黄昏

**《天净沙·秋思》马致远·元**
```
夕阳西下，断肠人在天涯。
```
> 适合黄昏、秋天、思乡

**《登乐游原》李商隐·唐**
```
向晚意不适，驱车登古原。
夕阳无限好，只是近黄昏。
```

**《暮江吟》白居易·唐**
```
一道残阳铺水中，半江瑟瑟半江红。
可怜九月初三夜，露似真珠月似弓。
```

## 八、雪景 ❄️

### 雪景经典

**《江雪》柳宗元·唐**
```
千山鸟飞绝，万径人踪灭。
孤舟蓑笠翁，独钓寒江雪。
```
> 适合雪景、孤寂感

**《雪梅》卢梅坡·宋**
```
梅须逊雪三分白，雪却输梅一段香。
```

**《逢雪宿芙蓉山主人》刘长卿·唐**
```
日暮苍山远，天寒白屋贫。
柴门闻犬吠，风雪夜归人。
```

**《沁园春·雪》毛泽东·现代**
```
北国风光，千里冰封，万里雪飘。
望长城内外，惟余莽莽；
大河上下，顿失滔滔。
```

## 九、雨天 🌧️

### 春雨

**《春夜喜雨》杜甫·唐**
```
好雨知时节，当春乃发生。
随风潜入夜，润物细无声。
```

**《清明》杜牧·唐**
```
清明时节雨纷纷，路上行人欲断魂。
```

### 秋雨

**《雨霖铃》柳永·宋**
```
多情自古伤离别，更那堪，冷落清秋节。
今宵酒醒何处？杨柳岸，晓风残月。
```

### 夜雨

**《夜雨寄北》李商隐·唐**
```
君问归期未有期，巴山夜雨涨秋池。
何当共剪西窗烛，却话巴山夜雨时。
```

## 十一、森林树木 🌲

**《竹里馆》王维·唐**
```
独坐幽篁里，弹琴复长啸。
深林人不知，明月来相照。

大雪压青松，青松挺且直。

```

### 梧桐

**《相见欢》李煜·南唐**
```
无言独上西楼，月如钩。
寂寞梧桐深院锁清秋。
```

## 十二、草原沙漠 🏜️

### 草原

**《敕勒歌》北朝民歌**
```
敕勒川，阴山下，
天似穹庐，笼盖四野。
天苍苍，野茫茫，
风吹草低见牛羊。
```

**《赋得古原草送别》白居易·唐**
```
离离原上草，一岁一枯荣。
野火烧不尽，春风吹又生。
```

### 沙漠

**《使至塞上》王维·唐**
```
大漠孤烟直，长河落日圆。
```

## 十三、四季 🗓️

### 春天

**《春晓》孟浩然·唐**
```
春眠不觉晓，处处闻啼鸟。
夜来风雨声，花落知多少。
```

### 夏天

**《小池》杨万里·宋**
```
泉眼无声惜细流，树阴照水爱晴柔。
小荷才露尖尖角，早有蜻蜓立上头。
```

**《山行》杜牧·唐**
```
远上寒山石径斜，白云生处有人家。
停车坐爱枫林晚，霜叶红于二月花。
```

### 冬天

**《梅花》王安石·宋**
```
墙角数枝梅，凌寒独自开。
遥知不是雪，为有暗香来。
```

## 十四、田园乡村 🏡

### 田园诗意

**《归园田居》陶渊明·晋**
```
种豆南山下，草盛豆苗稀。
晨兴理荒秽，带月荷锄归。
道狭草木长，夕露沾我衣。
衣沾不足惜，但使愿无违。
```

**《过故人庄》孟浩然·唐**
```
绿树村边合，青山郭外斜。
开轩面场圃，把酒话桑麻。
待到重阳日，还来就菊花。
```

## 十六、通用救场金句 🆘

当你不确定是什么风景时，这些万金油诗句：

| 场景 | 金句 |
|-----|------|
| 不知道该说什么 | "江山如画，一时多少豪杰" —— 苏轼 |
| 很壮观 | "造化钟神秀，阴阳割昏晓" —— 杜甫 |
| 很美但说不出 | "此景只应天上有，人间能得几回观" |
| 想装深沉 | "寄蜉蝣于天地，渺沧海之一粟" —— 苏轼 |
| 拍照发圈 | "春风得意马蹄疾，一日看尽长安花" —— 孟郊 |
| 看日出 | "日出而作，日入而息" ——《庄子》 |
| 看日落 | "夕阳无限好，只是近黄昏" —— 李商隐 |
| 看到大海 | "海纳百川，有容乃大" ——《孟子》 |
| 看到月亮 | "但愿人长久，千里共婵娟" —— 苏轼 |
| 看到花 | "年年岁岁花相似，岁岁年年人不同" |

## 十八、九年级义务教育必背诗词 📚

按年级分类整理，涵盖小学1-6年级 + 初中7-9年级全部必背篇目。

---

### 小学1-2年级

**《咏鹅》骆宾王·唐**
```
鹅，鹅，鹅，曲项向天歌。
白毛浮绿水，红掌拨清波。
```

**《江南》汉乐府·汉**
```
江南可采莲，莲叶何田田。
鱼戏莲叶间。
鱼戏莲叶东，鱼戏莲叶西，
鱼戏莲叶南，鱼戏莲叶北。
```

**《风》李峤·唐**
```
解落三秋叶，能开二月花。
过江千尺浪，入竹万竿斜。
```

**《画》王维·唐**
```
远看山有色，近听水无声。
春去花还在，人来鸟不惊。
```

**《古朗月行（节选）》李白·唐**
```
小时不识月，呼作白玉盘。
又疑瑶台镜，飞在青云端。
```

**《悯农（其二）》李绅·唐**
```
锄禾日当午，汗滴禾下土。
谁知盘中餐，粒粒皆辛苦。
```

**《池上》白居易·唐**
```
小娃撑小艇，偷采白莲回。

夫死战场子在腹，妾身虽存如昼烛。

妾家高楼连苑起，良人执戟明光里。
知君用心如日月，事夫誓拟同生死。
还君明珠双泪垂，恨不相逢未嫁时。
```

**《十五从军征》汉乐府·汉**
```
十五从军征，八十始得归。
道逢乡里人："家中有阿谁？"
"遥看是君家，松柏冢累累。"
兔从狗窦入，雉从梁上飞。
中庭生旅谷，井上生旅葵。
舂谷持作饭，采葵持作羹。
羹饭一时熟，不知贻阿谁。
出门东向看，泪落沾我衣。
```

**《白头吟》卓文君·汉**
```
皑如山上雪，皎若云间月。
闻君有两意，故来相决绝。
今日斗酒会，明旦沟水头。
躞蹀御沟上，沟水东西流。
凄凄复凄凄，嫁娶不须啼。
愿得一心人，白头不相离。
竹竿何袅袅，鱼尾何簁簁。
男儿重意气，何用钱刀为。
```

八百里分麾下炙，五十弦翻塞外声，沙场秋点兵。

**《西江月·夜行黄沙道中》辛弃疾·宋**

欲说还休，却道"天凉好个秋"！

**《水调歌头》苏轼·宋**
```
明月几时有？把酒问青天。
不知天上宫阙，今夕是何年？
我欲乘风归去，又恐琼楼玉宇，高处不胜寒。
起舞弄清影，何似在人间。
转朱阁，低绮户，照无眠。
不应有恨，何事长向别时圆？
人有悲欢离合，月有阴晴圆缺，此事古难全。
但愿人长久，千里共婵娟。
```

**《念奴娇·赤壁怀古》苏轼·宋**
```
大江东去，浪淘尽，千古风流人物。
故垒西边，人道是，三国周郎赤壁。
乱石穿空，惊涛拍岸，卷起千堆雪。
江山如画，一时多少豪杰。
遥想公瑾当年，小乔初嫁了，雄姿英发。
羽扇纶巾，谈笑间，樯橹灰飞烟灭。
故国神游，多情应笑我，早生华发。
人生如梦，一尊还酹江月。
```

料峭春风吹酒醒，微冷，山头斜照却相迎。
回首向来萧瑟处，归去，也无风雨也无晴。

为报倾城随太守，亲射虎，看孙郎。
酒酣胸胆尚开张，鬓微霜，又何妨！

会挽雕弓如满月，西北望，射天狼。
