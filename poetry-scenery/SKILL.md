# 诗意风景描述 Skill

让你面对美景时，不再只会说"我草真美"，而能随口抛出经典诗词，瞬间提升逼格。

## 核心功能

1. **识别风景类型** - 根据照片内容或用户描述，判断是山水、瀑布、大海、花草等
2. **匹配经典诗词** - 从诗词库中找出最匹配的诗句
3. **多维度输出** - 诗词 + 作者 + 解读 + 场景应用
4. **朋友圈文案** - 提供适合发朋友圈/小红书的优雅文案

## 使用场景

| 场景 | 示例输入 | 效果 |
|-----|---------|------|
| 旅行拍照 | 发一张瀑布照片 | 告诉你《望庐山瀑布》 |
| 发朋友圈 | 去看海了 | 甩出《春江花月夜》 |
| 写小红书 | 拍了很多花 | 推荐《江畔独步寻花》 |
| 装逼时刻 | 被美景震住了 | 救场金句来了 |

## 诗词匹配原则

1. **意象匹配** - 优先匹配照片中最突出的意象（山、水、云、花等）
2. **情感契合** - 考虑景色的氛围（壮阔、婉约、静谧、热烈）
3. **知名度优先** - 选择大多数人都听过的经典诗句
4. **长短适中** - 避免太长的诗，适合朋友圈展示

## 进阶技巧

- 如果有多首合适的诗，按匹配度递进推荐
- 可以要求"更小众的诗"或"更出名的诗"
- 可以问"这首诗怎么念/什么意思"
- 可以说"来首英文诗"或"来首日语俳句"
