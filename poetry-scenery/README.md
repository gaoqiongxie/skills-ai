# 诗意风景描述 📸✨

> 面对美景，不再只会说"我草真美"

## 功能简介

这是一个帮你用诗词描述风景的AI Skill。当你旅行拍照、发朋友圈、写小红书时，只需上传照片或描述看到的风景，AI会为你匹配最合适的古今诗词，让你的分享瞬间提升N个档次。

## 核心能力

- 📷 **智能识别** - 自动识别风景类型（山水/瀑布/花海/日落...）
- 📜 **诗词匹配** - 从100+首经典诗词中找出最贴切的
- ✍️ **文案生成** - 提供朋友圈/小红书优雅文案
- 🗣️ **装逼救场** - 面对美景不会说话时的救命金句

## 使用示例

### 场景1：拍到瀑布

**你：** 发一张瀑布照片

**AI输出：**
```
📜 《望庐山瀑布》李白·唐
日照香炉生紫烟，遥看瀑布挂前川。
飞流直下三千尺，疑是银河落九天。

🔹 朋友圈文案：
"飞流直下三千尺，疑是银河落九天"
（配图：瀑布照）
```

### 场景2：去西湖看荷花

**你：** "去看荷花了，怎么发朋友圈？"

**AI输出：**
```
📜 《晓出净慈寺送林子方》杨万里·宋
毕竟西湖六月中，风光不与四时同。
接天莲叶无穷碧，映日荷花别样红。

🔹 朋友圈文案：
"接天莲叶无穷碧，映日荷花别样红"
（配图：西湖荷花）
```

### 场景3：登泰山

**你：** "登顶了，看日出，怎么发？"

**AI输出：**
```
📜 《望岳》杜甫·唐
会当凌绝顶，一览众山小。

🔹 朋友圈文案：
"会当凌绝顶，一览众山小"
（配图：泰山日出）
```

## 进阶玩法

- **"来个小众的"** - 推荐不那么烂大街的诗
- **"来首英文诗"** - 给你找外国诗
- **"来首俳句"** - 日本俳句也很美
- **"这首诗什么意思"** - AI给你讲解

## 触发方式

直接说你看到了什么风景或上传照片：
- "拍了张瀑布照片"
- "去海边玩了，怎么发朋友圈？"
- "看到月亮了，好圆"
- "这是什么花？发朋友圈"
