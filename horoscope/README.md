# 星座解签

> 抽一张星象签，看看今天运势如何？仅供娱乐！

---

## 🎯 这是什么

一个趣味星座运势 + 抽签的娱乐Skill。

---

## 🚀 使用方法

```
"抽一张签"
"天秤座今天运势"
"白羊座本周运势"
"星座配对"
"水逆怎么办"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 你的星座 | 12星座之一 | 天秤座 |
| 时间范围 | 今日/本周/本月 | 本周 |
| 想问的事 | 事业/感情/财运 | 感情 |

---

## 📦 输出内容

1. **运势解读** - 根据星座性格分析
2. **抽签服务** - 随机抽取星象签
3. **趣味内容** - 星座配对、塔罗牌

---

## 💡 功能特点

- 🔮 每日运势解读
- 🎴 趣味抽签
- 🌟 星座性格分析
- ⚠️ 娱乐性质，开心就好
