# 会议纪要整理助手

> 混乱的会议记录？交给我，帮你整理成结构清晰的正式纪要！

---

## 🎯 这是什么

一个帮你把混乱会议记录整理成专业会议纪要的Skill。

---

## 🚀 使用方法

```
"帮我整理会议纪要"
"会议记录太乱了，帮我梳理下"
"把这段话整理成会议纪要格式"
```

---

## 📦 输入/输出

**输入**：原始会议记录（文字/录音转文字）

**输出**：结构化会议纪要
- 会议基本信息
- 讨论议题
- 决策事项
- 待办任务（带负责人+截止时间）
- 风险提示

---

## 💡 功能特点

- 📋 自动分层结构化
- ✅ 区分已决定/待讨论
- 👤 提取负责人和截止时间
- 📊 生成待办表格

---

## ⚡ 使用场景

- 项目评审会
- 周例会
- 需求评审
- 复盘会
- 一对一沟通

---
