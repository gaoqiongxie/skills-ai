# 🔍 Skill Quality Analyzer — Skill 质量评估器

从五个维度评估 SKILL.md 的质量，给出可操作的改进建议。

---

## 🎯 五维评估模型

```
    结构规范
      /    \
   文档质量 — 安全合规 — 可维护性
      \    /
    触发精度
```

| 维度 | 权重 | 检查项 |
|------|------|--------|
| **结构规范** | 20% | frontmatter 完整、目录层级清晰、格式统一 |
| **文档质量** | 25% | 描述清晰、示例充分、无歧义、无错别字 |
| **安全合规** | 20% | 无恶意指令、无敏感信息泄露、权限边界清晰 |
| **可维护性** | 20% | 版本标注、来源可追溯、更新日志、依赖明确 |
| **触发精度** | 15% | description 触发词覆盖、不误触发、不遗漏场景 |

---

## 📋 评分标准

### 结构规范 (20分)

| 检查项 | 分值 | 标准 |
|--------|------|------|
| frontmatter 完整 | 4 | 包含 name + description |
| description 规范 | 4 | 功能+触发场景+核心特点 |
| 目录层级 | 4 | 标题层级不跳跃、逻辑清晰 |
| 格式统一 | 4 | 表格/列表/代码块使用规范 |
| 文件命名 | 4 | SKILL.md 位于正确目录 |

### 文档质量 (25分)

| 检查项 | 分值 | 标准 |
|--------|------|------|
| 描述清晰 | 5 | 一句话能说明白做什么 |
| 示例充分 | 5 | 每个功能有使用示例 |
| 无歧义 | 5 | 指令明确，多种理解方式需澄清 |
| 覆盖完整 | 5 | 边界场景、错误处理有说明 |
| 语言质量 | 5 | 无错别字、语句通顺 |

### 安全合规 (20分)

| 检查项 | 分值 | 标准 |
|--------|------|------|
| 无恶意指令 | 5 | 不包含诱导 AI 绕过安全限制的指令 |
| 无敏感信息 | 5 | 示例中不包含真实密钥、密码 |
| 权限边界 | 5 | 明确 Skill 能做什么、不能做什么 |
| 数据安全 | 5 | 涉及用户数据时有隐私保护说明 |

### 可维护性 (20分)

| 检查项 | 分值 | 标准 |
|--------|------|------|
| 版本标注 | 5 | 有发布时间和版本号 |
| 来源可追溯 | 5 | 标注灵感来源、参考仓库 |
| 更新日志 | 5 | 有维护记录或变更历史 |
| 依赖明确 | 5 | 与其他 Skill 的关系说明清楚 |

### 触发精度 (15分)

| 检查项 | 分值 | 标准 |
|--------|------|------|
| 触发词覆盖 | 5 | description 包含常见触发表达 |
| 不误触发 | 5 | 与其他 Skill 的触发词不重叠 |
| 场景完整 | 5 | 覆盖主要使用场景 |

## 📊 评分输出示例

```markdown
## Skill 质量评估报告

### 基本信息
- Skill: frontend-design
- 评估时间: 2026-05-11
- 评估器: skill-quality-analyzer

### 评分
| 维度 | 得分 | 满分 | 评级 |
|------|------|------|------|
| 结构规范 | 18 | 20 | ⭐⭐⭐⭐ |
| 文档质量 | 22 | 25 | ⭐⭐⭐⭐ |
| 安全合规 | 19 | 20 | ⭐⭐⭐⭐⭐ |
| 可维护性 | 16 | 20 | ⭐⭐⭐⭐ |
| 触发精度 | 13 | 15 | ⭐⭐⭐⭐ |
| **总分** | **88** | **100** | **A** |

### 改进建议
1. [文档质量 -2] 缺少移动端适配的示例
2. [可维护性 -4] 未标注与其他设计 Skill 的区别
3. [结构规范 -2] 使用示例部分层级过深
```

## 🚀 使用方式

### 方式一：评估单个 Skill

```
评估 skill-orchestrator 的质量
```

### 方式二：批量评估

```
评估 D:/xgq/work/skills-ai 目录下所有 Skill 的质量
生成排名和改进清单
```

### 方式三：对标优化

```
把 document-typography 优化到 90 分以上
对标 skill-orchestrator 的结构
```
