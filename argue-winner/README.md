# 吵架赢对面

> 被客户骂、被甩锅、被PUA...怎么回击才不吃亏？告诉我场景，帮你想话术！

---

## 🎯 这是什么

一个帮你应对职场冲突场景的Skill。

不是教你骂人，是教你既不失专业又不吃亏！

---

## 🚀 使用方法

```
"客户骂我了，怎么回？"
"被领导甩锅了，怎么怼回去？"
"开会时被批评了怎么办？"
"同事老让我免费帮忙"
"怎么怼回去？"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 场景 | 什么情况 | 客户投诉 |
| 对方说了什么 | 具体内容 | 骂我产品垃圾 |
| 你的身份 | 什么岗位 | 售后客服 |

---

## 📦 输出内容

1. **应对话术** - 根据场景生成
2. **不同力度版本** - 温和/中等/强硬
3. **注意事项** - 雷区别踩

---

## 💡 功能特点

- 💬 针对7大职场冲突场景
- 🎯 提供不同力度的话术
