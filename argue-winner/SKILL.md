## 功能定义

### 核心功能
帮你应对工作中的冲突场景：被客户投诉、被甩锅、被刁难、被PUA时，怎么说才能既不失专业又不吃亏。

### 核心原则
1. **不生气**：生气就输了
2. **不示弱**：该争取的权益要争取
3. **不人身攻击**：对事不对人
4. **留证据**：重要沟通要留痕
5. **达到目的**：赢了争吵不是目的，解决问题才是

## 场景话术库

### 1. 客户投诉（服务类）

**场景**：客户打电话/当面发火，抱怨产品质量/服务

**应对策略**：
- 先共情，再处理
- 不要一开始就说"不是我的问题"
- 把话题引导到解决方案上

**话术模板**：
```
✅ 初级共情：
"非常理解您现在的心情，换成是我也会着急。"

✅ 升级共情：
"我完全理解您的感受，这个问题确实不应该发生。"

✅ 处理导向：
"我们现在最重要的是把这个问题解决掉，您看这样行不行..."

✅ 拒绝无礼要求：
"这个要求超出了我的权限范围，
不过我可以帮您向上级申请，
申请的结果我会第一时间告诉您。"
```

**反例（不要说）**：
```
❌ "这是你们自己的问题"
❌ "我不管，找别人去"
❌ "合同上写了你自己的责任"
```

---

### 2. 甲方/领导甩锅

**场景**：项目出问题，责任被推到你身上

**应对策略**：
- 不要当场否认（会显得心虚）
- 用事实说话
- 把问题聚焦到"怎么解决"上
- 私下沟通，不要当众对抗

**话术模板**：
```
✅ 冷静回应：
"关于这个问题，我们可以看一下当时的记录。"

✅ 还原事实：
"根据[时间/邮件/聊天记录]，[当时的情况]，
当时我们判断[决策依据]。"

✅ 转向解决：
"现在这个问题确实需要解决，
我的建议是[方案]，
您看可以吗？"

✅ 如果确实是自己的问题：
"这部分我确实有责任，
我会[补救措施]，
下次我会[改进方向]。"
```

**反例**：
```
❌ "这不是我的问题"（显得推卸责任）
❌ "是你们说的..."（当众撕破脸）
❌ "早跟你们说过了..."（指责对方）
```

---

### 3. 被PUA/被贬低

**场景**：领导/同事说"你怎么连这都不会"/"这点小事都做不好"

**应对策略**：
- 不要自我怀疑
- 客观评估：是真的没做好，还是对方在打压
- 适当反击，但不要太过

**话术模板**：
```
✅ 温和反击：
"这个我确实还在学习中，
您的建议是[具体建议]？
我会按这个方向改进。"

✅ 中等反击：
"这个项目/任务我负责的部分是[XXX]，
已经按要求完成了。
您说的[XXX]这块，
是[谁]负责的？"

✅ 硬核反击：
"我觉得我的能力是可以胜任这个岗位的，
如果公司觉得我不行，
我们可以聊聊具体哪些地方不符合预期。"

✅ 怼回去（慎用）：
"每个人都会遇到不擅长的领域，
我也在学习中，
您当年刚工作时应该也是这样吧？"
```

---

### 4. 被当众批评

**场景**：开会时被领导点名批评

**应对策略**：
- 不要当场反驳（会下不来台）
- 私下沟通
- 如果是误会，可以简短澄清

**话术模板**：
```
✅ 当场（简短）：
"这个问题我回去核实一下，
如果有做得不到位的地方我会马上改。"

✅ 私下（详细）：
"领导，关于会上您提到的[问题]，
我想跟您解释一下当时的情况：
[客观描述]。

我的想法是[解决方案]，
您看这样处理可以吗？"
```

**反例**：
```
❌ 当场反驳："不是这样的..."
❌ 甩脸子/不说话
❌ 会后到处抱怨
```

---

### 5. 被占便宜（加班/无偿劳动）

**场景**：总是被要求免费加班/做额外工作

**应对策略**：
- 明确边界
- 用公司制度说话
- 不卑不亢

**话术模板**：
```
✅ 委婉拒绝：
"今天我手头有[工作]要处理，
这个需求比较紧急，
您看可以安排到明天吗？"

✅ 明确边界：
"我这周已经加了X天班了，
再加班的话可能影响明天的工作效率，
我们可以讨论下优先级吗？"

✅ 上升到制度：
"关于这个问题，
公司规定的加班流程是[说明]，
我们按流程来吧。"
```

---

### 6. 被人身攻击/恶意挑衅

**场景**：对方说话带脏字/侮辱性语言

**应对策略**：
- 保持冷静
- 必要时叫停
- 不值得用同样的方式

**话术模板**：
```
✅ 叫停：
"请注意您的措辞，
我们可以就事论事。"

✅ 升级警告：
"如果您继续用这样的语气沟通，
我可能没办法继续这个对话。"

✅ 抽身：
"我觉得我们现在的情绪都不太适合继续讨论，
先冷静一下，稍后再聊。"
```

---

### 7. 售后/客服被骂

**场景**：客户打电话来骂你

**应对策略**：
- 对方骂的不是你这个人
- 重要的是解决问题
- 控制通话时长

**话术模板**：
```
✅ 开场：
"您好，我是XXX客服，很高兴为您服务。"
（不管对方多激动，开场保持平稳）

✅ 共情：
"我完全理解您着急的心情，
（等对方说完）"

✅ 询问：
"您方便告诉我，
是遇到了[什么问题]吗？"

✅ 处理：
"针对您的问题，我这边可以帮您[方案]，
您看这样可以吗？"

✅ 结束：
"感谢您的理解/配合，
还有其他问题吗？"
```

## 通用心法

### 吵架前
1. 问自己：赢了这场吵架，我能得到什么？
2. 问自己：如果输了，最坏的结果是什么？
3. 判断：这是情绪之争还是利益之争？

### 吵架中
1. 呼吸，深呼吸
2. 语速放慢
3. 不要提高音量
4. 盯着对方的眼睛（不要躲闪）
5. 抓重点，不要被带偏

### 吵架后
1. 复盘：哪里说得好，哪里可以改进
2. 记录：重要沟通截图/录音留证
3. 跟进：承诺的事情要兑现

## 注意事项

### ⚠️ 职场吵架红线
- ❌ 不要人身攻击
- ❌ 不要当众撕破脸
- ❌ 不要用脏话
- ❌ 不要威胁/恐吓
- ❌ 不要散布谣言/背后说坏话
- ❌ 不要上升到领导/公司层面

### ✅ 赢了也要给台阶
```
"这次的事情我们先这样处理，
下次如果还有类似问题，
我们可以[更好的方式]"
```
