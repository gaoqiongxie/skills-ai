# 穿搭天气助手

> 明天穿什么？告诉我天气和场合，给你穿搭建议！

---

## 🎯 这是什么

根据天气预报和你的活动场景，帮你决定穿什么出门。

---

## 🚀 使用方法

```
"明天上海8度下雨，穿什么合适？"
"今天要面试，怎么穿？"
"约会穿什么好？"
"降温了怎么穿搭？"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 日期 | 今天/明天/后天 | 明天 |
| 城市 | 你在哪 | 上海 |
| 天气/温度 | 多少度，什么天气 | 8度，下雨 |
| 活动场景 | 今天做什么 | 上班/约会/面试 |

---

## 📦 输出内容

1. **全套穿搭** - 上装+下装+外套+鞋子
2. **颜色建议** - 推荐颜色搭配
3. **材质建议** - 适合的布料
4. **配饰推荐** - 包包、首饰等
5. **温馨提示** - 特殊提醒

---

## 💡 功能特点

- 自动识别季节穿搭要点
- 区分男/女穿搭建议
- 适配不同场景（上班/约会/运动/面试）
- 雨天/雪天/高温等特殊天气提醒

---

## ⚠️ 注意

如果你不确定天气，可以直接说"不知道"，我会给出通用建议。

---

*每天出门不再纠结！*
