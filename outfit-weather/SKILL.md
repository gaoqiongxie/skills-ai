## 功能定义

### 核心功能
根据明天/当天的天气预报、活动场景，给你最合适的穿搭建议。

### 输入参数
- **日期**: 今天/明天/后天（可选，自动获取）
- **城市**: 你所在的城市
- **天气**: 温度、湿度、是否下雨/下雪、有风（可选，自动获取）
- **活动场景**: 上班/约会/逛街/运动/户外/聚会/面试/等
- **个人偏好**: （可选）喜欢穿的风格

---

## 天气与穿搭对照表

### 按温度分层

| 温度 | 穿搭建议 |
|-----|---------|
| 30°C+ | 短袖短裤/裙子、透气材质、防晒 |
| 25-30°C | 短袖/薄长袖、轻薄裤子/裙子 |
| 20-25°C | 长袖T恤、衬衫、薄外套 |
| 15-20°C | 薄毛衣/卫衣、薄外套、牛仔裤 |
| 10-15°C | 毛衣、厚外套、秋裤 |
| 5-10°C | 厚毛衣/羽绒服内胆、加绒裤子 |
| 0-5°C | 羽绒服、厚毛衣、围巾手套 |
| 0°C以下 | 羽绒服+保暖内衣、帽子围巾全副武装 |

### 按天气状况

| 天气 | 注意事项 |
|-----|---------|
| 下雨 | 带伞、穿防滑鞋、避免白鞋 |
| 下雪 | 防滑鞋、厚袜子、外套防水 |
| 大风 | 避免穿太宽松的、带帽子 |
| 雾霾 | 戴口罩、穿容易清洗的 |
| 艳阳高照 | 防晒、墨镜、遮阳帽 |

## 场景穿搭模板

### 上班通勤

```
【周一至周五上班】
基础款：衬衫/T恤 + 西装裤/直筒裤 + 小皮鞋/乐福鞋
外套：轻薄风衣/西装外套（空调房用）
包：电脑包/托特包

进阶：增加配饰（丝巾、胸针、手表）
```

### 约会穿搭

```
【甜系约会】
裙装：碎花裙 + 小白鞋/玛丽珍
外套：针织开衫
配饰：耳钉、项链

【酷系约会】
裤装：高腰裤 + 修身T恤 + 靴子
外套：皮夹克/西装外套
配饰：墨镜、金属项链
```

### 逛街出行

```
【舒适逛街】
上装：T恤/卫衣（宽松款）
下装：牛仔裤/运动裤
鞋：运动鞋/平底鞋
包：帆布包/双肩包

重点：舒适为主，方便试衣服！
```

### 运动健身

```
【室内健身】
上装：运动bra/T恤（吸汗材质）
下装：运动短裤/ leggings
鞋：运动鞋
配件：发带、运动水壶、毛巾

【户外跑步】
注意保暖层（可脱卸）
夜跑：穿反光装备
```

### 面试/商务

```
【正式商务】
西装套装（深色：黑/藏青）
衬衫（浅色）
皮鞋（黑色/棕色）
配饰：领带（可选）、手表

【一般商务】
西装裤 + 衬衫
或：西裙 + 针织衫
鞋：皮鞋/高跟鞋（女）
```

## 季节穿搭要点

### 春天（3-5月）
- 早晚温差大 → 多层穿搭（外套+内搭）
- 柳絮/花粉 → 戴口罩、备好过敏药
- 推荐：风衣、针织开衫、卫衣、衬衫

### 夏天（6-8月）
- 防晒！防晒！防晒！
- 透气材质优先
- 推荐：短袖、连衣裙、凉鞋、防晒衣

### 秋天（9-11月）
- "春捂秋冻"但别太冻
- 叠穿是王道
- 推荐：风衣、毛衣、卫衣、衬衫+外套

### 冬天（12-2月）
- 保暖内衣很重要
- 层层叠加
- 推荐：羽绒服、厚毛衣、围巾帽子手套

## 男生/女生分别建议

### 男生
- 基础款：白T恤、白衬衫、深色牛仔裤、休闲裤
- 鞋子：小白鞋、休闲皮鞋、运动鞋
- 配色：黑白灰 + 一个亮色点缀

### 女生
- 基础款：T恤、衬衫、针织衫、牛仔裤、裙子
- 鞋子：小白鞋、靴子、乐福鞋、高跟鞋
- 配色：莫兰迪色系、安全又高级
