# 职场反PUA话术 AI Skill

> 帮你用智慧和底气，优雅地怼回去

## 功能简介

这是一个职场反PUA话术生成工具。当你遭遇老板/上级的精神打压、道德绑架、画饼、甩锅等职场PUA场景时，只需描述对方的原话或你的遭遇，AI将为你生成有力的回击话术。

## 触发场景

当你有类似需求时会用到这个Skill：

- 老板说"你怎么连这点事都做不好？"
- 领导画饼："好好干，以后这个位置就是你的"
- 被道德绑架："大家都在加班，你好意思走？"
- 出了问题被甩锅："这事怎么搞成这样？"
- 被模糊评价："你的表现不够好"
- 情绪勒索："我对你真的很失望"
- 其他任何让你感到被精神控制的职场场景

## 使用方法

直接告诉AI你的遭遇，例如：

- "老板让我周末加班，我说有事，他说'大家都在拼，就你特殊？'怎么怼回去？"
- "领导说'换成别人早就完成了，你怎么这么差'，怎么回？"
- "老板承诺的年终奖没发，还说'你要理解公司'，怎么办？"

AI会根据你的场景，生成3-5条可选话术，按力度递进排列，并附上话术解析和后续行动建议。

## 核心原则

| 原则 | 说明 |
|-----|------|
| 逻辑清晰 | 用事实和数据说话，不情绪化 |
| 态度坚定 | 明确边界，不轻易让步 |
| 不卑不亢 | 尊重但不卑微，有力但不攻击 |
| 保护自己 | 留存证据，明确责任归属 |

## 安装方法

将 `anti-pua` 文件夹放入 `~/.workbuddy/skills/` 目录下，重启 WorkBuddy 即可使用。
