# 职场反PUA话术 Skill

本Skill提供专业的职场反PUA话术生成服务，帮助用户在面对职场精神控制时保持清醒、优雅且有力地回应。

## 核心原则

1. **逻辑清晰** - 用事实和数据说话，不情绪化
2. **态度坚定** - 明确边界，不轻易让步
3. **不卑不亢** - 尊重但不卑微，有力但不攻击
4. **保护自己** - 留存证据，明确责任归属

## 常用话术类型

详见 `references/话术库.md`，涵盖以下场景：

| 场景类型 | 典型话术策略 |
|---------|-------------|
| 打压贬低型 | 引用具体数据，反问求证 |
| 画饼型 | 追问具体承诺，索要书面确认 |
| 道德绑架型 | 强调权责对等，拒绝模糊责任 |
| 甩锅型 | 明确责任边界，要求证据 |
| 超额压榨型 | 引用劳动法，明确工作边界 |
| 模糊评价型 | 要求具体化描述，提供量化标准 |

## 使用方法

1. **用户输入**：描述老板/上级说了什么，或描述遭遇的职场场景
2. **分析场景**：识别PUA类型和话术陷阱
3. **生成回应**：提供3-5条可选话术，附讲解
4. **建议策略**：提醒注意事项和后续行动
