# HTML PPT 生成器 Skill

在 Agent 工作流中，一句话生成专业级 HTML 演示文稿。

---

## 🎨 核心能力

| 维度 | 数量 | 说明 |
|------|------|------|
| 主题 | **36套** | 商务、科技、极简、活泼、暗黑等风格 |
| 完整Deck模板 | **15套** | 开箱即用的整套演示模板 |
| 页面布局 | **31种** | 封面、目录、内容页、对比、数据等 |
| 动画效果 | **47种** | 27种CSS动画 + 20种Canvas特效 |

---

## 🚀 使用方式

### 基本用法

```
用户：帮我做一个"Q1业绩总结"PPT，5页，商务风格
```

```
用户：做一个产品介绍PPT，主题科技感，需要有数据对比页
```

```
用户：做个汇报材料，内容是：[粘贴文字]
```

### 带详细要求

```
用户：帮我做PPT
主题：Python 学习入门
页数：8页
风格：简洁活泼
动画：需要有入场动画
```

## 📋 标准页面类型

| 布局类型 | 适用场景 |
|---------|---------|
| 封面页 | 标题、副标题、作者 |
| 目录页 | 章节导航 |
| 标题+要点 | 列举核心观点 |
| 左图右文 | 图文混排 |

## 🎨 主题风格指南

### 商务类
- **Corporate Blue** — 深蓝+白，正式商务汇报
- **Executive Dark** — 深色背景，高端感
- **Clean Minimal** — 极简白，苹果风

### 科技类
- **Tech Purple** — 紫色渐变，科技感
- **Neon Cyber** — 霓虹配色，未来感
- **Data Viz** — 数据可视化专用

### 活泼类
- **Colorful Pop** — 多彩活泼，教育培训
- **Pastel Soft** — 马卡龙色，柔和清新
- **Bold Gradient** — 大胆渐变，创意展示

## ⚙️ 生成流程

```
1. 解析需求
   ↓ 主题 / 页数 / 风格 / 内容

2. 选择模板
   ↓ 匹配最适合的主题 + 布局组合

3. 填充内容
   ↓ 自动排版文字、数据、图表

4. 添加动画
   ↓ 根据风格选择合适的动画效果

5. 输出 HTML
   ↓ 单文件，浏览器直接打开展示
```

---

## 📝 输出规范

- **格式**：单个 `.html` 文件，可直接用浏览器打开
- **尺寸**：16:9 宽屏（1920×1080）
- **字体**：内嵌字体，无需安装
- **导出**：浏览器打印 → 另存为 PDF
