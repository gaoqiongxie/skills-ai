# 外卖选择困难症终结者

> 纠结吃什么？告诉我，帮你快速做决定！

---

## 🎯 这是什么

一个帮你解决"今天吃什么"选择困难症的Skill。

输入你的口味、人数、预算，秒出推荐！

---

## 🚀 使用方法

```
"吃什么？"
"两个人，100块预算，想吃辣的"
"一个人简简单单吃点"
"推荐个外卖选择"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 口味 | 想吃什么类型 | 川菜、日料、火锅、面食 |
| 人数 | 几个人吃 | 1人、2人、3-4人 |
| 预算 | 价格范围 | 50以内、100左右 |
| 特殊要求 | 忌口/偏好 | 不要辣、素食、减肥 |
| 心情 | 此刻的心情 | 想补补、想吃甜的 |

---

## 📦 输出内容

1. **主推荐** - 最终选择 + 理由
2. **备选方案** - 1-2个替代
3. **搭配建议** - 主菜+配菜+饮品
4. **预算参考** - 价格范围

---

## 💡 使用场景

- 中午不知道吃什么
- 两个人约会选餐厅
- 周末在家点外卖
- 请客不知道推荐什么
