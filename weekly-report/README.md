# 周报生成器

> 每周写周报头疼？告诉我你做了什么，帮你生成专业周报！

---

## 🎯 这是什么

一个帮你快速生成简洁、专业、有亮点周报的Skill。

---

## 🚀 使用方法

```
"帮我写周报"
"本周做了登录模块、修了一个bug、开了评审会"
"生成本周工作总结"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 本周工作 | 做了什么事 | 做了XX功能、修bug |
| 下周计划 | 要做什么 | 开始XX模块 |
| 问题/需要支持 | （可选） | 需要设计稿 |
| 量化数据 | （可选） | 提升了30%效率 |

---

## 📦 输出内容

1. **本周工作总结** - 分条列项，突出成果
2. **下周工作计划** - 具体可执行
3. **需协助事项** - 有问题主动提出
4. **可选数据指标** - 量化你的贡献

---

## 💡 功能特点

- 📊 数据化呈现你的工作成果
- ✍️ 专业书面语言
- ⏱️ 3秒生成，告别周报焦虑
- 🔄 支持日报、周报、月报

---

## ⚡ 使用场景

- 每周五写周报
- 项目节点写总结
- 试用期写工作汇报
- 跳槽时整理工作经历
