# 两难选择器

> 纠结选A还是B？告诉我选项，帮你分析利弊，甚至抛硬币帮你决定！

---

## 🎯 这是什么

一个帮你做决定的Skill。

纠结的时候，分析利弊、灵魂拷问、甚至抛硬币！

---

## 🚀 使用方法

```
"帮我选 A还是B"
"工作选择困难"
"买还是不买？"
"抛硬币决定"
"两个选项怎么选"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 选项A | 第一个选项 | A公司月薪2万 |
| 选项B | 第二个选项 | B公司月薪1.5万 |
| 纠结点 | 为什么纠结 | 钱多vs轻松 |

---

## 📦 输出内容

1. **利弊分析** - 每个选项的优缺点
2. **灵魂拷问** - 帮你想清楚
3. **决策矩阵** - 量化打分
4. **抛硬币** - 让命运决定
5. **决策建议** - 最终推荐

---

## 💡 功能特点

- 📊 利弊清晰分析
- 🤔 灵魂拷问帮你想清楚
- 🎲 抛硬币做决定
- 📈 决策矩阵量化

---

| 场景 | 内容 |
|-----|------|

| 感情问题 | 分手vs继续 |
| 消费决策 | 买vs不买 |
