- **版本**: 1.0.0

---

## 功能定义

### 核心功能
帮你分析纠结的选择，给出决策建议，甚至可以"抛硬币"帮你决定。

### 输入
- 选项A和B的具体内容
- 纠结的原因（可选）
- 你的顾虑（可选）

### 输出
- 利弊分析
- 决策建议
- 或者抛硬币决定

## 选择决策框架

### 1. 利弊分析法

```
┌─────────────────────────────────┐
│         【利弊分析】             │
├─────────────────────────────────┤
│                                 │
│  选项A：[名称]                  │
│  ✓ 优点1：                     │
│  ✓ 优点2：                     │
│  ✗ 缺点1：                     │
│  ✗ 缺点2：                     │
│                                 │
│  选项B：[名称]                  │
│  ✓ 优点1：                     │
│  ✓ 优点2：                     │
│  ✗ 缺点1：                     │
│  ✗ 缺点2：                     │
│                                 │
└─────────────────────────────────┘
```

### 2. 决策矩阵法

```
┌─────────────────────────────────┐
│         【决策矩阵】             │
├─────────────────────────────────┤
│                                 │
│  评估因素    │ 权重 │ A │ B │  │
│  ────────────┼────┼───┼───┤  │
│  短期收益    │ 20% │ 7 │ 8 │  │
│  长期收益    │ 30% │ 8 │ 9 │  │
│  风险大小    │ 25% │ 6 │ 4 │  │
│  喜欢程度    │ 25% │ 9 │ 6 │  │
│  ────────────┼────┼───┼───┤  │
│  加权得分    │ 100%│7.6│6.8│  │
│                                 │
│  🏆 推荐：[选项A]              │
│                                 │
└─────────────────────────────────┘
```

### 3. 灵魂拷问法

```
【灵魂拷问】

❓ 如果选A，1年后我会后悔吗？
❓ 如果选B，1年后我会后悔吗？

❓ 哪个选择更接近我想要的生活？

❓ 如果这是我最后一次做选择，我会选？

❓ 10年后回看，哪个选择让我更骄傲？
```

### 4. 田忌赛马法

```
【田忌赛马策略】

把每个选项的优势列出来，
看哪个选项的优势对你更重要。

例如：
A公司：钱多、大厂光环
B公司：成长快、氛围好

如果你看重成长 → B更好
如果你看重简历 → A更好
```

## 常见选择场景

### 1. 工作选择

**场景**：A工作 vs B工作

```
【决策要点】

✓ 选A的情况：
- 工资更高
- 平台更大
- 更稳定
- 离家近

✓ 选B的情况：
- 成长更快
- 更感兴趣
- 团队更好
- 更有前景

📊 决策建议：
问自己：
1. 你现在最需要什么？（钱/成长/稳定）
2. 5年后你想成为什么样的人？
3. 哪个选择让你更兴奋？

💡 没有绝对的对错，
只有更适合你现在的选择。
```

### 2. 感情选择

**场景**：分手还是继续

```
⚠️ 注意：感情问题没有标准答案

【自我提问】

❓ 你是舍不得这个人，
   还是舍不得这段感情？

❓ 这段关系让你变得更好，
   还是更差了？

❓ 如果朋友遇到同样的情况，
   你会怎么建议他们？

❓ 1年后、5年后回头看，
   你会后悔什么？

💭 有时候分开是为了更好的相遇
💭 有时候坚持是值得的
💭 跟着心走，但也要脑子清醒
```

### 3. 消费选择

**场景**：买还是不买

```
【决策框架】

❓ 这是"想要"还是"需要"？
- 需要 → 买
- 想要 → 继续问

❓ 买了之后会后悔吗？
- 不会 → 可以买
- 会 → 不买

❓ 这个东西能提升你的生活质量吗？
- 能 → 考虑买
- 不能 → 算了

❓ 如果是1元，你会买吗？
- 会 → 说明你真的想要
- 不会 → 说明只是便宜

💡 买得起的，不一定买得起
💡 钱不是消失了，只是换了一种方式陪着你
```

### 4. 人生方向

**场景**：大城市 vs 小城市

```
【大城市】

✓ 机会多、资源多
✓ 见识广、成长快
✓ 工资相对高
✗ 压力大、生活成本高
✗ 节奏快、竞争激烈
✗ 可能孤独

【小城市】

✓ 生活成本低、压力小
✓ 离家人近
✓ 生活节奏慢
✗ 机会相对少
✗ 资源有限
✗ 可能无聊

📊 决策建议：

看阶段：
- 年轻人：大城市机会多
- 有家庭：小城市更稳定

看性格：
- 喜欢挑战 → 大城市
- 喜欢安稳 → 小城市

💡 没有最好的选择
💡 只有适合现在的选择
💡 选了就去全力以赴
```

## 抛硬币决策法

### 标准抛硬币

```
🎲 【抛硬币】

正面：A
反面：B

我帮你抛...

🪙 硬币在空中旋转...

🪙 结果：正面！

🎉 硬币说：选A！

━━━━━━━━━━━━━━━━━━━━━━━━

💭 但等等...
硬币抛出那一刻，
你是不是心里已经有了答案？

如果你"希望"是正面，
说明你想选A。

这不是硬币在决定，
是你自己在决定。
```

### 改良版抛硬币

```
🎲 【三选一抛硬币】

选项A、B、C

硬币：A=数字面，B=花色，C=立起来

━━━━━━━━━━━━━━━━━━━━━━━━

🪙 结果：A！

🎉 硬币说：选A！

━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 注意：硬币只是帮你做决定
真正重要的选择，还是要自己思考
```

## 决策小技巧

### 10-10-10法则

```
【10-10-10法则】

10分钟后：这个决定重要吗？
   → 大多数都不重要

10个月后：这个决定还重要吗？
   → 有些会变得重要

10年后：这个决定还有影响吗？
   → 大多数都没影响了

💡 很多事没有我们想的那么重要
💡 专注于真正重要的选择
```

### 最坏情况法则

```
【最坏情况法则】

问自己：

如果选A，最坏的情况是？
   → [写出来]

如果选B，最坏的情况是？
   → [写出来]

哪个最坏情况更能接受？
   → 选那个
```

### 逆向思考法

```
【逆向思考】

问自己：
- 如果不选A，我会后悔吗？
- 如果不选B，我会后悔吗？

哪个后悔更强烈？
   → 选那个不让自己后悔的
```

## 使用示例

### 示例

**用户**：该去A公司（月薪2万，工作累）还是B公司（月薪1.5万，工作轻松）

**AI输出**：
```
🎯 【两难选择分析】

你的选择：
A公司：月薪2万，但工作很累
B公司：月薪1.5万，工作轻松

━━━━━━━━━━━━━━━━━━━━━━━━

【利弊对比】

选项A（月薪2万/累）
✓ 优点：
  - 钱多5000/月
  - 可能有更好发展
  - 大厂光环
✗ 缺点：
  - 累
  - 可能没时间享受

选项B（月薪1.5万/轻松）
✓ 优点：
  - 工作轻松
  - 有时间休息/学习
  - 生活质量可能更高
✗ 缺点：
  - 钱少5000
  - 可能发展有限

━━━━━━━━━━━━━━━━━━━━━━━━

【灵魂拷问】

❓ 你现在最缺的是什么？
   - 缺钱 → 选A
   - 缺时间 → 选B

❓ 5年后你想成为什么样的人？
   - 职场精英 → A
   - 工作生活平衡 → B

❓ 如果A和B工资一样，你选哪个？
   - 选A → 其实你想去A
   - 选B → 其实你想选B

━━━━━━━━━━━━━━━━━━━━━━━━

【决策建议】

基于你的描述，给你两个方向：

🚀 如果你：
- 刚毕业/还年轻
- 缺钱/要攒钱
- 想快速成长
→ 推荐：A公司
（年轻时拼一拼，以后有的是时间休息）

🧘 如果你：
- 工作多年累了
- 有家庭要照顾
- 更看重生活
→ 推荐：B公司
（钱够用就行，身体和生活更重要）

━━━━━━━━━━━━━━━━━━━━━━━━

【抛硬币决定】

🎲 让硬币帮你决定？

正面：A公司
反面：B公司

你选哪面？

━━━━━━━━━━━━━━━━━━━━━━━━

💭 无论选哪个，
记住：选择不是终点，
你随时可以调整方向。
```
