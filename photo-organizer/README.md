# 照片整理助手 📸

> 告别"照片黑洞"，让相册井井有条

## 功能简介

这是一个帮你整理手机和电脑照片的AI Skill。无论照片再多再乱，AI都能帮你：
- 按时间线归类
- 按人物主角分组
- 发现重复照片
- 生成整理方案

## 核心能力

| 功能 | 说明 |
|-----|------|
| 📅 时间识别 | 自动识别照片拍摄时间 |
| 👤 人物识别 | 识别照片中的人物主角 |
| 🔄 重复检测 | 发现相似/重复照片 |
| 📊 智能统计 | 统计各类照片分布 |
| 📁 方案生成 | 生成整理文件夹结构 |
| 🧹 清理建议 | 指出可清理项 |

## 使用方式

1. 告诉我照片在哪里（手机/电脑路径）
2. 描述你的需求（整理/清理/分类）
3. AI生成整理方案
4. 按方案执行整理
