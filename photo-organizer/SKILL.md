# 照片整理助手 Skill

帮你告别"照片黑洞"，让相册井井有条。

## 核心功能

### 1. 时间线整理
- 识别照片拍摄时间
- 按年/月/日归类
- 发现同一地点不同时间的照片

### 2. 人物主角分类
- 识别照片中的人物
- 按人物主角分组
- 标记"单人照"、"合照"、"风景照"

### 3. 相似照片处理
- 识别连拍/重复照片
- 建议保留最佳一张
- 标记HDR/RAW等变体

### 4. 整理报告生成
- 生成文件夹结构建议
- 输出整理清单
- 统计各类照片数量

📂 宝宝成长/

## 使用场景

| 场景 | 需求 |
|-----|------|
| 手机内存不足 | 分析照片占用，找出可清理项 |
| 相册太乱 | 整理归类，建立秩序 |
| 宝宝成长记录 | 按年龄整理，方便回顾 |
| 旅行照片 | 按地点+时间整理 |
| 清理重复 | 发现并处理相似照片 |
| 备份整理 | 分类整理后备份 |

## 整理策略

### 按时间线
```
📂 年份/
    └── 📂 月份/
        └── 📂 具体日期_事件/
```

### 按人物
```
📂 宝宝/
📂 家庭/
📂 朋友/
📂 自己/
```

### 按地点
```
📂 上海/
📂 杭州/
📂 三亚/
📂 日本/
```

### 按用途
```
📂 生活记录/
📂 工作素材/
📂 证件照/
📂 截图/
```

## 操作流程

1. **提供照片信息**
   - 照片路径或描述照片内容
   - 说出想整理的需求

2. **AI分析**
   - 识别时间、地点、人物
   - 发现规律和重复

3. **生成方案**
   - 输出整理结构建议
   - 给出具体操作步骤

4. **执行整理**
   - 按需帮你移动/归类照片
   - 生成整理报告

## 注意事项

- 整理前建议先备份
- 重要照片单独标记
- 截图和生活照分开存放
- RAW和JPG保留一份即可
