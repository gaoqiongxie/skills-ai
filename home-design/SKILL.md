## 装修风格库

### 1. 北欧风（北欧极简）

**特点**：
- 白色为主，原木色点缀
- 简洁干净，线条简单
- 大量留白
- 绿植点缀

**适合人群**：
- 年轻人
- 小户型
- 喜欢简约

**颜色搭配**：
```
主色：白色、米白
辅色：浅灰、原木色
点缀：绿色、黄色

配色公式：
白 + 原木 + 绿植
```

**设计要点**：
- 大量收纳，保持整洁
- 选择简洁家具
- 布艺软装提升温馨感
- 绿植增加生机

### 2. 日式原木风

**特点**：
- 自然木色为主
- 温暖、禅意
- 榻榻米、推拉门
- 收纳强大

**适合人群**：
- 喜欢安静
- 需要多功能空间
- 小户型

**颜色搭配**：
```
主色：原木色、米白
辅色：竹色、纸色
点缀：灰色、绿色

配色公式：
原木 + 白 + 绿
```

### 3. 现代简约

**特点**：
- 黑白灰为主
- 线条感强
- 金属/玻璃材质
- 无过多装饰

**适合人群**：
- 都市白领
- 追求品质
- 大户型

**颜色搭配**：
```
主色：白色、灰色
辅色：黑色、金属色
点缀：蓝色、橙色

配色公式：
灰 + 黑 + 金
```

### 4. 轻奢风

**特点**：
- 大理石、金属元素
- 低饱和度配色
- 质感强
- 低调奢华

**适合人群**：
- 有一定预算
- 追求品质
- 成熟人士

**颜色搭配**：
```
主色：灰色、白色
辅色：金色、黑色
点缀：墨绿、酒红

配色公式：
灰白 + 金 + 绿植
```

### 5. 奶油风

**特点**：
- 奶白色、奶油色
- 圆弧形元素
- 温柔治愈
- 少女心

**适合人群**：
- 女性
- 喜欢温馨感
- 小清新

**颜色搭配**：
```
主色：奶油白、米色
辅色：浅粉、浅黄
点缀：金色、绿色

配色公式：
奶白 + 粉 + 金
```

### 6. 工业风

**特点**：
- 水泥、砖墙、金属
- 深色调
- 复古感
- 粗犷个性

**适合人群**：
- 男性
- 喜欢个性
- Loft/大空间

**颜色搭配**：
```
主色：黑色、灰色
辅色：棕色、金属色
点缀：红色、黄色

配色公式：
黑 + 灰 + 棕
```

### 7. 中式风格

**特点**：
- 红木色、黑色
- 对称布局
- 山水画、屏风
- 文化底蕴

**适合人群**：
- 喜欢传统文化
- 大户型
- 成熟人士

**颜色搭配**：
```
主色：红木色、深棕
辅色：黑色、金色
点缀：米白、绿色

配色公式：
深棕 + 黑 + 金
```

## 空间灵感

### 客厅

**设计要点**：
- 沙发是核心
- 电视背景墙不要太复杂
- 收纳很重要
- 灯光营造氛围

**收纳方案**：
```
电视柜收纳
沙发背景墙收纳
角落置物架
茶几收纳
```

### 卧室

**设计要点**：
- 床是核心
- 灯光要柔和
- 衣柜要够用
- 避免过度装饰

**配色建议**：
```
主色：浅色系
辅色：同色系深色
点缀：1-2个亮色抱枕
```

### 厨房

**设计要点**：
- U型/L型利用率高
- 台面要够用
- 灯光要亮
- 收纳要做好

**收纳技巧**：
```
吊柜收纳
调料拉篮
水槽下收纳
墙壁挂件
```

### 卫生间

**设计要点**：
- 干湿分离
- 镜子要大
- 收纳要够
- 防滑要重视

**扩容技巧**：
```
壁挂式马桶
镜柜收纳
壁龛收纳
淋浴房玻璃隔断
```

### 小户型重点

**扩容技巧**：
1. **开放式设计**
   - 客餐厅一体
   - 开放式厨房
   
2. **收纳至上**
   - 定制柜子
   - 高柜利用空间
   
3. **浅色系**
   - 白色为主
   - 视觉扩容
   
4. **多功能家具**
   - 折叠餐桌
   - 沙发床
   - 榻榻米

## 颜色搭配指南

### 安全配色公式

```
1. 同色系深浅搭配
   深灰 + 浅灰 + 白
   
2. 对比色点缀
   白 + 灰 + 一抹绿
   
3. 暖色系温馨
   米白 + 原木 + 橙
   
4. 冷色系清爽
   白 + 蓝 + 灰
```

### 配色雷区

```
❌ 不要超过3种主色
❌ 不要五颜六色
❌ 不要饱和度太高的颜色
❌ 不要每个房间颜色都不一样
```
