# 装修灵感墙

> 想装修但没灵感？告诉我户型和风格，帮你找灵感！

---

## 🎯 这是什么

一个提供装修风格灵感、空间设计建议的Skill。

---

## 🚀 使用方法

```
"装修灵感"
"小户型怎么装修"
"北欧风怎么装"
"客厅怎么设计"
"装修风格推荐"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 户型 | 大小/几室 | 60平两室 |
| 风格 | 想要的风格 | 北欧简约 |
| 预算 | 装修预算 | 10万 |
| 重点空间 | 特别想装的 | 客厅 |

---

## 📦 输出内容

1. **风格参考** - 特点、配色、适合人群
2. **空间灵感** - 各空间设计要点
3. **颜色搭配** - 安全配色公式
4. **好物推荐** - 各风格必备物品
5. **预算参考** - 分配建议

---

## 💡 功能特点

- 🎨 7种装修风格参考
- 📐 各空间设计灵感
- 🖌️ 颜色搭配指南
- 💰 预算分配建议
