## 🎯 核心内容

### ⚠️ 十大戒律
1. 不要过度抽象
2. 先跑通，再优化
3. 写测试是必须的
4. 读报错比问 AI 快
5. 先自己体验
6. 避免超级代理
7. 上下文要具体
8. 警惕 AI 幻觉引用
9. 代码 review 不能省
10. 保持人工判断力

### 🛠️ 调试流程
```
读报错 → 定位 → 思考 → 提问 → 验证 → 理解
```
