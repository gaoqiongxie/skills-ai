## 功能定义

### 核心功能
本Skill用于生成以20世纪40-90年代中国北方农村为背景、以"奶奶"与"孙女（或第一人称'我'）"为核心的短篇小说（2000-4000字）。

### 擅长领域
- 捕捉特定节气/节日中的传统食物制作
- 民俗活动描写
- 祖孙深情叙事
- 语言朴实温润，细节饱满
- 情感克制而悠长

### 适用主题
- 传统吃食（蒸野菜、烙饼、炒豆、蒜薹炒肉、榆钱窝窝、红薯叶、玉米饼等）
- 民俗仪式（画粮圈、炒蝎子爪、祭灶、守岁、上坟、扫房等）
- 祖孙关系与代际传承
- 时代变迁下的乡村记忆

---

## 人物设定

### 奶奶 👵
- **年龄**: 60-80岁
- **身份**: 农村老年女性
- **性格**: 勤劳、隐忍、不善言辞
- **技能**: 擅长各种农活和传统吃食制作
- **外貌**: 常穿蓝布褂子、黑围裙，头发花白，手粗糙
- **台词风格**: 简短、口语化，动作多于言语

### "我" / 滚滚 👧
- **年龄**: 7-10岁女童（或成年后回忆的第一人称）
- **性格**: 天真、好奇、黏奶奶
- **角色**: 故事的观察者与情感载体
- **视角**: 第一人称或第三人称有限视角

### 其他配角（按需启用）
- 姥爷、父母、二叔二婶
- 村里的动物：大鹅、土狗、公鸡（可沿用"村霸"等角色设定）

---

## 场景设定

### 时间维度
- **首选**: 特定节气/节日
  - 春节、二月二、清明、谷雨、立夏、端午、中秋、腊月
- **次选**: 四季日常（春种秋收、夏忙冬闲）

### 空间维度
- 北方农村典型环境：
  - 土坯房 / 砖瓦房
  - 灶屋（老式柴火灶）
  - 院子（枣树、榆树、菜园）
  - 打谷场
  - 田野 / 河滩

---

## 核心事件类型

### 类型一：制作传统食物
从原料采集到烹饪的全过程描写：
- **采集**: 挖野菜、捋榆钱、割蒜薹、采香椿
- **处理**: 择、洗、切、和面
- **烹饪**: 蒸、烙、炒、炸

### 类型二：民俗仪式
- 画粮圈、炒蝎子爪、祭灶、守岁、上坟
- 扫房、蒸年糕、炸焦叶子

### 必备要素
每个故事必须包含**一个具体动作**的详细描写：
> 示例："奶奶蹲下来，抓一把草木灰从指缝漏出，在地上画出一个圆"

---

## 情感结构

### 明线（表层）
奶奶对"我"的宠爱：
- 做吃的
- 带玩
- 教规矩
- 默默承担

### 暗线（深层）
时代变迁下的隐痛：
- 贫穷
- 计划生育
- 外出打工
- 老去与死亡
- 代际传承的失落

---

## 语言风格规范

### 总体要求
- **朴实白描**: 少用形容词堆砌，多用动词和名词
- **感官饱满**: 视觉、嗅觉、听觉、触觉全覆盖
- **句式特点**: 以短句为主，节奏舒缓，留白多

### 感官描写要点
| 类型 | 示例 |
|-----|------|
| 视觉 | 颜色、光影、动作 |
| 嗅觉 | 柴火味、油香、野菜清甜 |
| 听觉 | 锅铲声、风声、鸡叫 |
| 触觉 | 粗糙的手、滚烫的锅沿 |

### 方言点缀（适度使用）
薅、捋、醒面、炕熟、笸箩、秫秸等

### 禁忌
- ❌ 现代网络用语
- ❌ 华丽辞藻堆砌
- ❌ 直白说教
- ❌ 过度煽情

### 结尾要求
- 以成年后的追忆收束
- 或某种循环（如"我"也画粮圈、做蒸菜）
- 形成闭环，余味悠长

## 生成流程

### 第一步：确定主题与节气
1. 询问用户想写的节气/节日
2. 或由用户自由输入关键词（如"榆钱""炒面""腊八粥"）
3. 如用户未指定，推荐以下选项：
   - 二月二（炒蝎子爪 + 画粮圈）
   - 清明（蒸野菜 / 蒜薹炒肉）
   - 谷雨（香椿）
   - 立夏（煮鸡蛋）
   - 端午（包粽子）
   - 中秋（烙糖饼）
   - 腊月（蒸年糕 / 炸焦叶）

### 第二步：确定核心事件
根据主题自动匹配典型事件：

| 节气/节日 | 核心事件 |
|----------|---------|
| 二月二 | 炒蝎子爪、画粮圈 |
| 清明 | 挖荠菜/灰灰菜/苋菜、蒸野菜、蒜薹炒肉 |
| 谷雨 | 采香椿、炸香椿鱼 |
| 立夏 | 煮立夏蛋、编蛋兜、称人 |
| 端午 | 包粽子、挂艾草 |
| 中秋 | 烙糖饼、供月亮 |
| 腊月 | 扫房、蒸年糕、炸焦叶子 |
| 春节 | 包饺子、守岁、放鞭炮 |

### 第三步：生成故事结构

```
开篇
├── 时间、环境交代
└── 奶奶早起 / "我"被唤醒

采集/准备
├── 田野/灶屋细节
└── 原料准备过程

制作过程
├── 手法描写
├── 火候变化
└── 气味细节

品尝/完成
├── 祖孙互动
└── 明线情感

暗线植入
├── 时代背景暗示
└── 隐痛表达

结尾
├── 回到当下
└── 或成年后回忆，形成闭环
```

### 第四步：语言检查清单

- [ ] 无现代网络用语
- [ ] 无华丽辞藻堆砌
- [ ] 至少3处感官细节描写
- [ ] 结尾有余韵，不直白说教
- [ ] 方言使用适度（不过度）
- [ ] 短句为主，节奏舒缓

## 调用示例

### 示例1
**用户**: 帮我写一篇3000字的小说，主题是立夏煮鸡蛋。

**Skill执行**:
1. 确认节气：立夏
2. 核心事件：煮立夏蛋、编蛋兜、称人
3. 生成故事：奶奶早起 → 选蛋 → 煮蛋 → 编蛋兜 → 给"我"挂蛋 → 讲立夏习俗
4. 暗线：奶奶年轻时的艰辛 / "我"长大后离乡

### 示例2
**用户**: 写一个关于奶奶和榆钱的故事

**Skill执行**:
1. 节气：清明前后（榆钱盛开）
2. 核心事件：捋榆钱 → 做榆钱窝窝
3. 生成故事：春天天奶奶带"我"上树 → 捋榆钱 → 洗榆钱 → 和面 → 蒸窝窝
4. 暗线：村庄变迁 / 奶奶老去

### 示例3
**用户**: 二月二要到了，帮我想一个故事

**Skill执行**:
1. 推荐：炒蝎子爪 + 画粮圈
2. 核心事件：奶奶用面做蝎子爪形状 → 炒熟 → 画粮圈祈丰收
3. 生成故事：节日清晨 → 奶奶和面 → 做蝎子爪 → 院子里画粮圈 → "我"帮忙
4. 暗线：农耕时代的祈愿 / 村庄记忆
