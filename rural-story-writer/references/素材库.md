# 乡土叙事小说素材库

包含：节气故事模板、食物制作细节、民俗仪式描写、方言词汇表

## 一、节气故事模板

### 二月二 · 龙抬头

**时间背景**: 正月二十之后，春耕开始
**天气**: 乍暖还寒，院子里还有残雪
**核心食物**: 蝎子爪（面食）、春饼
**民俗活动**: 画粮圈、炒蝎子爪、剃龙头

**奶奶的动作描写**:
```
奶奶从面缸里舀出一瓢白面，在瓷盆里兑上温水。
她把面揉成一个拳头大小的剂子，搓成细长条，
用刀切成指甲盖大小的小剂子。
然后奶奶拿起一个小剂子，两手一搓，
就弯成了蝎子的形状——
身子是圆的，尾巴翘起来，带着两个钳子。
```

**画粮圈的描写**:
```
奶奶蹲在地上，从灶膛里掏出一把草木灰。
她把灰攥在掌心，用拇指和食指捏着，
让灰从指缝漏下来，在地上画出一个圆。
圆圈里，奶奶再画一个十字，
把粮食撒在十字的位置。
"圈住了，"奶奶说，"粮食就跑不了了。"
```

**暗线方向**:
- 农耕时代对丰收的祈愿
- 粮食的珍贵与匮乏
- 村庄里外出打工的年轻人

### 清明

**时间背景**: 公历四月四或五日，春耕时节
**天气**: 乍暖还寒，有时有雨
**核心食物**: 蒸野菜（荠菜、灰灰菜、苋菜）、蒜薹炒肉
**民俗活动**: 上坟、踏青、插柳

**挖野菜的描写**:
```
清明前后，地里的荠菜最嫩。
奶奶扛着䦆头，我挎着竹篮，跟在她身后。
荠菜贴地长，叶子锯齿状，开小白花。
奶奶用䦆头尖撬，我蹲下来用手抠。
筐里的荠菜越来越多，带着泥土的腥气，
混着青草的清甜。
```

**蒸野菜的描写**:
```
奶奶把荠菜择了又择，抖掉泥土，
用井水洗了三遍，摊在篦子上晾干。
然后她抓一把白面，拌进菜里，
让每片叶子都裹上薄薄的面粉。
蒸锅里的水咕嘟咕嘟开了，
奶奶把篦子放进去，盖上锅盖。
灶膛里的火烧得正旺，
我闻到一股清甜的气味从锅盖缝里冒出来。
```

**蒜薹炒肉的描写**:
```
奶奶从咸菜缸旁边的挂绳上取下一块腊肉，
切成薄薄的片，在热锅里煸出油。
蒜薹是年后菜园子里长的，细细长长的，
奶奶用刀背拍了一下，切成寸段。
蒜薹下锅，"刺啦"一声，
肉香和蒜香混在一起，冒起白烟。
```

**暗线方向**:
- 计划生育时代的紧张
- 去世的亲人
- 父母不在身边的童年

### 谷雨

**时间背景**: 四月二十日前后，春天最后一个节气
**天气**: 温暖，雨水增多
**核心食物**: 香椿炒蛋、香椿鱼、谷雨茶
**民俗活动**: 祭海、禁蝎

**采香椿的描写**:
```
院子里那棵香椿树不高，但枝丫脆。
爷爷拿着带钩子的竹竿拧，我举着篮子接。
香椿叶子是紫红色的，边缘镶着绿边，
碰一下就掉，落在脖子里痒痒的。
奶奶说"香椿要趁嫩吃，
长老了就是臭椿。"
她把香椿芽切碎，打上两个鸡蛋，
搅成糊，在热油里摊成金黄的饼。
```

**暗线方向**:
- 爷爷的衰老或离去
- 村庄的变迁

### 立夏

**时间背景**: 五月五或六日，夏天开始
**天气**: 渐热
**核心食物**: 立夏蛋、烧嫩麦、豌豆糕
**民俗活动**: 称人、挂蛋兜

**煮立夏蛋的描写**:
```
立夏这天，奶奶早早煮了一锅鸡蛋。
鸡蛋是自家鸡下的，个头不大，壳发白。
奶奶在灶台边坐下，拿起一个鸡蛋
在手心里转几圈，
然后放进锅里，和茶叶、八角一起煮。
灶膛里烧的是去年攒的麦秆，
火苗软软的，舔着锅底。
鸡蛋煮熟了，奶奶捞出来放进凉水里激一下，
蛋壳上染了淡淡的茶色，还带着香味。
```

**编蛋兜的描写**:
```
奶奶拿出五色线，坐在门槛上编蛋兜。
红线、绿线、黄线、白线、黑线，
五种颜色的线在她手里翻来翻去，
不一会儿就编成了一个网袋。
奶奶把鸡蛋装进去，
系在我脖子上，说：
"立夏要称人，也要挂蛋，
小孩儿一个夏天不生病。"
```

**暗线方向**:
- 奶奶年轻时的艰辛
- "我"长大后离乡

### 端午

**时间背景**: 五月初五，夏天
**天气**: 渐热，蚊虫多
**核心食物**: 粽子、艾草糕、黄鳝
**民俗活动**: 包粽子、挂艾草、系五彩线

**包粽子的描写**:
```
端午节前一天，奶奶就开始包粽子了。
糯米是提前泡好的，胀得白白胖胖。
粽子叶是奶奶从河边采的苇叶，
一张张洗干净，在开水里烫软。
奶奶把两片叶子叠在一起，
卷成漏斗形，
舀一勺糯米放进去，
再塞进一颗红枣，
然后把叶子折过来，
用马兰草系紧。
粽子包好了，放进大锅里煮。
一煮就是一夜，
第二天早上，粽子的香味把整个村子都熏透了。
```

**暗线方向**:
- 离别（父母外出打工）
- 家庭的不完整

### 中秋

**时间背景**: 八月十五，秋收时节
**天气**: 凉爽
**核心食物**: 月饼、糖饼、水果
**民俗活动**: 供月亮、赏月

**烙糖饼的描写**:
```
中秋的月亮圆得像白面饼，
奶奶这天要烙糖饼。
她把发面揉成长条，切成剂子，
擀成薄薄的圆皮，
包上红糖和炒熟的芝麻，
再把口捏紧，按成饼状。
铁锅烧热了，奶奶在锅底抹一层薄油，
把糖饼放进去，
"刺啦"一声，饼底就焦了。
奶奶用铲子翻个面，
过一会儿，糖饼鼓起来，
用指头一按，软软的，熟了。
```

**供月亮的描写**:
```
天黑了，月亮升起来，
奶奶把糖饼、水果摆在院子里的小桌上。
她点了三炷香，插在香炉里，
对着月亮鞠了三个躬。
我站在旁边，也学着奶奶的样子，
奶奶说："月亮娘娘保佑咱家，
风调雨顺，粮食满仓。"
月光洒下来，照在糖饼上，
我馋得直流口水。
```

**暗线方向**:
- 团圆与分离
- 父母在远方

### 腊月 · 小年

**时间背景**: 腊月二十三
**天气**: 寒冬
**核心食物**: 祭灶糖、灶饼
**民俗活动**: 祭灶、扫房

**祭灶的描写**:
```
腊月二十三，奶奶要祭灶。
她把灶王爷的画像从墙上请下来，
画像上灶王爷和灶王奶奶并排坐着，
慈眉善目的。
奶奶在灶台上摆上祭灶糖，
糖瓜又甜又黏，粘牙。
她烧了三炷香，嘴里念叨：
"灶王爷，您老人家，
上天言好事，下界保平安。"
然后把灶王爷的画像和纸钱一起烧了，
火光映在奶奶脸上，
我看见她的眼眶湿湿的。
```

**暗线方向**:
- 一年到头的辛苦
- 对来年的祈愿

### 除夕/春节

**时间背景**: 腊月三十或正月初一
**天气**: 寒冬
**核心食物**: 饺子、年糕、焦叶子
**民俗活动**: 包饺子、守岁、放鞭炮

**包饺子的描写**:
```
除夕那天，全家人都围着灶台包饺子。
奶奶和面、擀皮，
我抢着学包，
把饺子包得歪歪扭扭，像小耳朵。
奶奶包的饺子站得整整齐齐，
像一排排小兵。
馅是萝卜猪肉的，
奶奶在肉馅里放了花椒水、香油、葱姜末，
搅得黏黏糊糊，香气扑鼻。
```

**守岁的描写**:
```
三十晚上要守岁。
奶奶在堂屋里烧起一堆火，
说是"熬百岁"。
我困得眼皮打架，
奶奶就给我讲故事。
她说她小时候，
有一年除夕下大雪，
把她家的草房都压塌了。
我听着听着，就睡着了。
第二天醒来，
枕头边放着奶奶给的压岁钱——
两毛崭新的票子。
```

**暗线方向**:
- 家庭团聚的珍贵
- 年味渐淡的失落
- 亲人老去

## 二、食物制作细节库

### 蒸菜系列

#### 榆钱窝窝
```
1. 捋榆钱：春天，榆树长出一串串榆钱。
   用竹竿绑上钩子，把枝条拧下来，
   一把一把往下撸。
   
2. 洗榆钱：榆钱里有小虫子和杂质，
   要用清水泡，漂洗几遍，
   然后摊开晾干。

3. 和面：榆钱里拌上玉米面，
   加一点白面，用手拌匀，
   让每片榆钱都裹上面粉。

4. 蒸：用手攥成窝窝头形状，
   或者直接铺在篦子上。
   大火烧开，蒸十五分钟。

5. 出锅：揭开锅盖，
   榆钱从紫绿色变成了深绿色，
   一股清甜的气味扑鼻而来。
```

#### 蒸野菜（荠菜/灰灰菜/苋菜）
```
1. 挖菜：清明前后是最佳时节。
   拿着小铲子去地里挖，
   贴着地皮割，根不要。

2. 择洗：把黄叶子、杂草择掉，
   抖掉泥土，在水里洗三遍。

3. 拌面：野菜晾干水分，
   撒上薄薄一层面粉，
   用手轻轻拌匀。

4. 蒸：篦子上铺一块笼布，
   把拌好的菜铺开，大火蒸。

5. 蘸料：大蒜捣成泥，加醋、香油。
```

#### 蒸槐花
```
1. 捋槐花：槐花盛开时，满树白花，
   用竹竿拧下来，一串串的。

2. 洗槐花：槐花里有小虫子，
   要用盐水泡一会儿，再洗。

3. 拌面：槐花晾干，拌上面粉，
   面粉要少，让每朵花都裹上一层。

4. 蒸：铺在篦子上，大火烧开，
   蒸五分钟，槐花就熟了。

5. 调味：槐花本身有甜味，
   可以不加蘸料，也可以加糖。
```

### 面食系列

#### 葱油饼
```
1. 和面：面粉加温水，揉成光滑的面团，
   醒半小时。

2. 做油酥：碗里放面粉，加热油，
   搅拌成糊状。

3. 擀饼：面团分成剂子，
   擀成薄皮，抹上油酥，
   撒上葱花和盐。

4. 卷起：从一头卷起，卷成筒状，
   再盘起来，按扁。

5. 烙：平底锅烧热，放油，
   把饼放进去，两面烙黄。

6. 出锅：饼皮酥脆，葱香扑鼻，
   用手一掰，层层分明。
```

#### 水烙馍
```
1. 和面：面粉加温水，揉成软面团。

2. 揪剂子：把面团揪成鸡蛋大小的剂子。

3. 擀皮：剂子擀成薄薄的圆皮，
   越薄越好，透光。

4. 烙：把薄饼放在热锅里，
   不用油，饼一受热就鼓起来，
   翻面，再鼓，就熟了。

5. 叠起：烙好的饼叠在一起，
   用笼布盖上，保持软乎。
```

#### 焦叶子（炸面食）
```
1. 和面：面粉加鸡蛋、糖、油，
   揉成硬面团。

2. 擀面：把面团擀成薄片，
   越薄越脆。

3. 切块：用刀切成菱形或长方形。

4. 炸：油烧开，把面片放进去，
   "刺啦"一声，面片就鼓起来，
   炸到金黄，捞出控油。

5. 冷却：凉了就酥了，
   一咬"咔嚓"响，满嘴香。
```

### 炒制系列

#### 炒蝎子爪（二月二）
```
1. 和面：白面加温水，揉成面团，
   面要硬一点，才好切。

2. 切剂子：把面团搓成细长条，
   切成指甲盖大小的小剂子。

3. 搓形：每个剂子搓成两头弯弯的形状，
   像蝎子的爪子。

4. 炒：锅里放沙子（或粗盐）加热，
   把蝎子爪倒进去翻炒，
   火不能太大，不然糊了。

5. 出锅：蝎子爪变成金黄色，
   又酥又脆，咬着"嘎嘣"响。
```

#### 炒豆子
```
1. 备料：黄豆或黑豆，捡去杂质。

2. 干炒：锅里不放油，
   把豆子倒进去，小火翻炒。

3. 听声：豆子开始"噼啪"响，
   声音越来越密。

4. 出锅：等声音停了，
   豆子变色，就熟了。

5. 特点：嘎嘣脆，越嚼越香。
```

## 三、民俗仪式描写库

### 画粮圈
```
时间：二月二早上
动作：
1. 奶奶天不亮就起来了
2. 从灶膛里掏出一把草木灰
3. 蹲在地上，把灰攥在掌心
4. 拇指和食指捏着，让灰漏下来
5. 画出圆圈，边画边念叨
6. 圆圈里画十字
7. 在十字上撒粮食（麦粒、玉米粒）
意义：祈求新一年粮食丰收

台词：
"圈住了，圈住了，粮食就跑不了了"
```

### 祭灶
```
时间：腊月二十三（小年）
动作：
1. 打扫灶台
2. 摆上祭灶糖（糖瓜）
3. 烧香，三炷
4. 对着灶王爷画像鞠躬
5. 烧纸钱和灶王爷画像
6. 放鞭炮

台词：
"灶王爷，您老人家，
上天言好事，下界保平安"
"二十三，糖瓜粘，
灶王爷，您别上天"
```

### 扫房
```
时间：腊月二十四
动作：
1. 把屋里东西都搬出来
2. 奶奶头上包着头巾
3. 用长杆绑上笤帚
4. 从屋顶扫到墙角
5. 扫掉一年的灰尘和晦气
6. 扫出的垃圾堆到路口烧掉

意义：辞旧迎新
```

### 上坟
```
时间：清明、农历十月初一
动作：
1. 准备供品：点心、水果、纸钱
2. 用铁锨添新土
3. 压纸（把纸钱压在坟头上）
4. 烧纸钱
5. 磕头
6. 说说话

台词：
"爷爷，俺来看你了"
"给你送钱来了，别舍不得花"
```

## 六、故事结尾模板

### 类型一：循环式
```
"后来我长大了，
离开村子很多年。
有一年春天，
我在超市里看到一盒榆钱，
犹豫了很久，还是买了。
回家蒸了一锅，
却怎么也蒸不出奶奶的味道。
那天晚上，我梦见自己又回到了老屋的院子里，
奶奶站在香椿树下，
对我招招手：'妮儿，来，帮你奶奶捋榆钱。'
"
```

### 类型二：追忆式
```
"奶奶去世很多年了。
老屋早就塌了，
院子里那棵香椿树也不知被谁刨了。
但每年清明，
我还是会梦见那个院子，
梦见奶奶站在灶台边，
把刚出锅的蒸菜端到我面前。
她笑着说：'妮儿，趁热吃。'
"
```

### 类型三：留白式
```
"后来我才明白，
奶奶那时候做蒸菜，
不是因为爱吃，
是因为地里没什么菜，
而白面也不够吃。
那些野菜和粗粮，
喂饱了我的童年，
也喂饱了那个年代的穷。
"
```

### 类型四：闭环式
```
"现在我也学会做蒸野菜了。
每次做的时候，
我都会想起奶奶的样子：
蹲在地里薅菜，
弯着腰择菜，
站在灶台前往篦子上铺菜。
女儿说我做的蒸菜比饭店的好吃，
我没告诉她，
那是我奶奶教我的。
她的奶奶没教过她。
"
```
