# 乡土叙事小说生成器（Rural China Story Writer）

> 以20世纪40-90年代中国北方农村为背景，写出有温度的祖孙故事

---

## 🎯 这是什么

一个专门生成乡土叙事短篇小说的Skill。

以"奶奶"与"孙女"为核心，用朴实温润的语言，捕捉传统食物制作、民俗活动中的祖孙深情。

---

## 📖 适用主题

### 传统吃食
蒸野菜、烙饼、炒豆、蒜薹炒肉、榆钱窝窝、红薯叶、玉米饼、香椿炒蛋、粽子、年糕、焦叶子、腊八粥……

### 民俗活动
画粮圈、炒蝎子爪、祭灶、守岁、上坟、扫房、贴春联、放鞭炮……

### 情感内核
- 祖孙深情（明线）
- 时代变迁下的隐痛（暗线）
- 代际传承与失落

## 🚀 使用方法

### 方式一：直接描述需求
```
"帮我写一篇3000字的乡土小说，主题是清明蒸野菜"
"写一个奶奶做葱油饼的故事"
"生成一个二月二的祖孙故事"
```

### 方式二：描述关键词
```
"榆钱"
"奶奶的灶台"
"1990年代的农村"
```

### 方式三：指定节气
```
"立夏的故事"
"我想看端午节的故事"
```

## ✨ 风格特点

### 语言
- ✅ 朴实白描，少用形容词
- ✅ 短句为主，节奏舒缓
- ✅ 方言点缀（薅、捋、醒面等）

### 感官
- 🔴 视觉：颜色、光影
- 🔴 嗅觉：柴火味、油香
- 🔴 听觉：锅铲声、风声
- 🔴 触觉：粗糙的手、滚烫的锅沿

### 结尾
- 余韵悠长
- 形成闭环（"我"也画粮圈/做蒸菜）
- 不直白说教

---

## 📝 生成流程

1. **确认主题** → 节气/节日 或 关键词
2. **匹配事件** → 食物制作 或 民俗仪式
3. **生成故事** → 开篇→采集→制作→品尝→暗线→结尾
4. **语言检查** → 无网络用语、有感官细节
