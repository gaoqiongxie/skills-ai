# 睡前故事

> 睡不着？闭上眼睛，让我给你讲个故事，陪你入睡~

---

## 🎯 这是什么

一个帮你入睡的睡前故事Skill。

温暖的童话、治愈的故事、ASMR式引导...

---

## 🚀 使用方法

```
"讲个睡前故事"
"睡不着"
"哄我睡觉"
"讲个童话"
"听个故事睡觉"
```
