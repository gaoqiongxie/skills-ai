## 功能定义

### 核心功能
生成各种风格的睡前故事，帮你放松入睡。

### 输入
- 你想要的风格
- 或者随机一个故事

### 输出
- 完整的故事
- 温柔的语调
- 帮助你入睡

## 故事模板

### 童话模板

```
【睡前故事】小兔子的星星

在一个遥远的地方，
有一片美丽的森林。

森林里住着一只小兔子，
她的名字叫朵朵。

朵朵有一双长长的耳朵，
和一个圆圆的尾巴。

有一天晚上，
朵朵躺在床上睡不着。

她数着窗外的星星：
一颗，两颗，三颗...

星星们闪闪发光，
像是在对朵朵眨眼睛。

"晚安，小星星"
朵朵轻轻地说

"晚安，小兔子"
星星们眨眨眼睛回答

朵朵慢慢闭上眼睛，
进入了甜甜的梦乡。

在梦里，
她和星星们一起跳舞，
度过了美好的夜晚。

🌙 晚安，亲爱的
好梦...
```

### 治愈故事模板

```
【睡前故事】一片叶子的旅行

从前，有一片小小的叶子，
长在一棵大树上。

春天，她刚刚探出头，
嫩绿嫩绿的，像刚出生的小生命。

夏天，她变成深绿色，
每天和朋友们一起唱歌跳舞。

有一天，风来了，
叶子们开始聊天。

"我要飞去哪里呢？"
叶子们好奇地问。

风说：
"你不用决定，
我会带你去最好的地方。"

于是，叶子离开了树枝，
在空中飘啊飘。

她飘过了高山，
飘过了河流，
飘过了一个个小村庄。

最后，她落在了一片土地上，
变成了泥土的一部分。

第二年春天，
在她变成泥土的地方，
长出了一朵小花。

🌸 小花对着天空微笑，
那是叶子的新生命。

我们每个人，
都像那片叶子，
在生命的旅程中，
最终都会找到自己的位置。

晚安，亲爱的
不管今天发生了什么，
明天又是新的一天。

好梦... 🌙
```

```
【助眠故事】云朵棉花糖

闭上眼睛，

飘啊飘的...

你什么都没想，
只是躺着，
只是飘着。

风轻轻地吹，
云朵慢慢地飘。

你越来越放松，
越来越困。

晚安...
好梦...
🌙
```

## 使用示例

### 示例1：想听一个童话

**用户**：讲个睡前故事

**AI输出**：
```
🌙 【睡前故事】小兔子的星星

从前，在一片美丽的森林里，
住着一只小兔子，名叫朵朵。

朵朵长着一身雪白的毛，
两只长长的耳朵，
和一双红红的眼睛。

一天晚上，
月亮挂在天空中，
星星们眨着眼睛。

朵朵躺在床上，
翻来覆去睡不着。

"也许数星星会有帮助"
朵朵想。

于是她开始数：
一颗星星，两颗星星，三颗星星...

数着数着，
朵朵发现有一颗星星特别亮。

"那颗星星，你为什么这么亮呢？"

星星眨眨眼说：
"因为我在守护你呀，
每个睡不着的孩子，
都有一颗星星在守护。"

朵朵笑了，
心里暖暖的。

"晚安，星星"
"晚安，小兔子"

朵朵闭上眼睛，
慢慢地，慢慢地，
进入了甜甜的梦乡。

🌙 晚安，亲爱的
好梦...
```

## 助眠技巧

### 配合故事使用

1. **灯光调暗**
2. **放下手机**
3. **深呼吸**
4. **听故事**
5. **闭眼想象**

### 快速入睡建议

- 📱 睡前1小时不看手机
- 🧘 做深呼吸（4-7-8呼吸法）
- 🛁 泡个热水澡
- 📖 听舒缓的故事
- 🛏️ 固定的睡眠时间
