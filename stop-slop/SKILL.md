# 🧹 Stop Slop — 去除 AI 味

识别并清除 AI 生成内容中的典型 "slop" 模式，输出自然、有力、像人写的文字。

---

## 🎯 什么是 AI Slop？

AI Slop 是指大模型生成内容中常见的、模式化的、缺乏真实感的表达痕迹。包括但不限于：

### 常见 Slop 模式

| Slop 类型 | 示例 | 修正 |
|-----------|------|------|
| **Throat-clearing 开头** | "在当今快速发展的数字化时代..." | 直接说事 |
| **Em-dash reveal** | "他不是普通人——他是超级英雄" | 避免戏剧化断句 |
| **过度道歉** | "抱歉，我可能说得不对，但是..." | 直接陈述观点 |
| **空洞总结** | "总之，这是一个复杂而多面的话题" | 给出具体结论 |
| **列表滥用** | 什么都列 3-5 条，即使不需要 | 合并为连贯段落 |
| **填充短语** | "值得注意的是..."、"必须强调的是..." | 删掉，直接说内容 |
| **AI 口头禅** | "让我们..."、"记住..."、"不要忘记..." | 改用陈述句 |
| **安全套话** | "两者各有优劣，取决于具体情况" | 给出明确判断 |
| **Emoji 过度** | 每段都加 emoji | 控制使用频率 |
| **段末升华** | 强行上升到人生哲理 | 就事论事 |

## 🛠️ 工作流程

### 第一步：识别 Slop

扫描输入内容，标记所有 slop 模式：

```
[SLOP-1]  throat-clearing: "在当今快速发展的..."
[SLOP-2]  填充短语: "值得注意的是"
[SLOP-3]  空洞总结: "总之，这是一个复杂的话题"
```

### 第二步：评分

给出 AI Slop 指数（1-10）：

| 分数 | 含义 |
|------|------|
| 1-2 | 几乎无 slop，像人写的 |
| 3-4 | 少量 slop，轻微机器味 |
| 5-6 | 明显 slop，典型 AI 输出 |
| 7-8 | 严重 slop，满屏套话 |
| 9-10 | 纯 slop，几乎全是模板 |

### 第三步：重写

去除所有 slop，保持原意，输出自然表达：

**原则：**
- ✅ 主动语态 > 被动语态
- ✅ 具体 > 抽象
- ✅ 简洁 > 冗长
- ✅ 直接说 > 铺垫绕弯
- ✅ 一个观点一段，不强行凑列表

## 💡 使用示例

### 示例1：去除职场文档 AI 味

**输入：**
```
在当今快速发展的商业环境中，团队协作的重要性不言而喻。
值得注意的是，有效的沟通能够显著提升工作效率。
然而，我们必须认识到，沟通过程中可能会出现各种挑战。
总之，建立良好的沟通机制是团队成功的关键。
```

**分析：**
- [SLOP-1] throat-clearing: "在当今快速发展的商业环境中"
- [SLOP-2] 填充短语: "值得注意的是"
- [SLOP-3] 填充短语: "然而，我们必须认识到"
- [SLOP-4] 空洞总结: "总之，建立良好的沟通机制是团队成功的关键"
- **AI Slop 指数: 8/10**

**输出：**
```
团队协作需要有效沟通。明确分工、定期同步、及时反馈，比开会时长更重要。
```

### 示例2：去除技术文档 AI 味

**输入：**
```
让我们来探讨一下这个函数的实现原理。首先，我们需要理解它的输入参数。
然后，让我们看看它是如何处理这些参数的。最后，不要忘记返回值的处理。
记住，代码的可读性非常重要！
```

**分析：**
- [SLOP-1] AI口头禅: "让我们来探讨一下"
- [SLOP-2] 列表滥用: "首先...然后...最后"
- [SLOP-3] AI口头禅: "不要忘记"
- [SLOP-4] AI口头禅: "记住...非常重要"
- **AI Slop 指数: 7/10**

**输出：**
```
这个函数接收用户 ID 和订单数据，校验后写入数据库，返回写入结果或错误码。
```

### 示例3：去除产品描述 AI 味

**输入：**
```
我们的产品不仅仅是一个工具——它是您工作效率的终极解决方案！
通过创新的算法和直观的界面，我们重新定义了用户体验。
无论您是初学者还是专家，都能轻松上手。
选择我们，选择成功！
```

**分析：**
- [SLOP-1] em-dash reveal: "不仅仅是...——它是..."
- [SLOP-2] 大词堆砌: "重新定义了用户体验"
- [SLOP-3] 万能受众: "无论您是初学者还是专家"
- [SLOP-4] 口号式结尾: "选择我们，选择成功"
- **AI Slop 指数: 9/10**

**输出：**
```
这个工具自动整理收件箱，按优先级分类邮件，每天节省约 20 分钟。
```
