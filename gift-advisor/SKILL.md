## 功能定义

### 核心功能
根据送礼对象、关系、预算、场合，推荐贴心又有创意的礼物方案。

### 输入参数
- **收礼人**: 爸爸/妈妈/男朋友/女朋友/闺蜜/同事/领导/老师/孩子/等
- **关系**: 亲密/一般/商务/长辈/晚辈
- **年龄**: 便于精准推荐
- **预算**: 50以内/100左右/200-500/500-1000/1000+
- **场合**: 生日/情人节/母亲节/父亲节/中秋/春节/升职/毕业/结婚/等
- **对方喜好**: （可选）已知偏好
- **禁忌**: （可选）不能送什么

## 收礼人分类推荐

### 送长辈（父母/爷爷奶奶/外公外婆）

| 预算 | 推荐礼物 |
|-----|---------|
| 100以内 | 保暖内衣、护手霜、足浴盆、营养品 |
| 100-300 | 按摩仪、血压计、丝巾、旅游券 |
| 300-500 | 智能手机/平板、品质茶叶、保健品 |
| 500+ | 旅游套餐、金饰、健康体检 |

**送礼要点**: 实用、品质、让他们享受生活

### 送同辈（闺蜜/兄弟/朋友）

| 预算 | 推荐礼物 |
|-----|---------|
| 100以内 | 香水小样、创意小物、零食礼包、书籍 |
| 100-300 | 护肤品、香薰蜡烛、蓝牙耳机、游戏皮肤 |
| 300-500 | 轻奢配饰、手表、体验券（密室/剧本杀） |
| 500+ | 电子产品、奢侈品小件、旅行 |

**送礼要点**: 投其所好、有趣、有仪式感

### 送孩子

| 预算 | 推荐礼物 |
|-----|---------|
| 100以内 | 绘本、文具、玩具、拼图 |
| 100-300 | 乐高、儿童手表、学习机、轮滑装备 |
| 300-500 | 自行车、电子琴、无人机、编程课 |
| 500+ | 手机/平板（需家长同意）、电脑 |

**送礼要点**: 有趣、能学习、家长认可

### 送领导/客户

| 预算 | 推荐礼物 |
|-----|---------|
| 200以内 | 茶叶、书签、高端办公用品 |
| 200-500 | 茶叶礼盒、酒类（需了解喜好）、摆件 |
| 500-1000 | 燕窝/海参、高端皮具、小众品牌 |
| 1000+ | 旅游卡、健康管理、高端定制礼品 |

**送礼要点**: 品质、品味、不过分贵重、不踩雷

## 场合分类推荐

| 场合 | 礼物方向 |
|-----|---------|
| 生日 | 投其所好 + 有仪式感包装 |
| 情人节 | 浪漫、情侣相关 |
| 母亲节 | 感恩、妈妈需要但不舍得买的 |
| 父亲节 | 实用、爸爸的兴趣爱好 |
| 中秋 | 月饼 + 养生类 |
| 春节 | 坚果礼盒、酒类、特产 |
| 升职 | 恭喜 + 品质礼物 |
| 毕业 | 纪念意义、职场相关 |
| 结婚 | 家电、床上用品、红包（最实用） |

## 创意礼物点子库

### 平价但有心意
- **手写信** + 照片书
- **DIY相册**/DIY蛋糕
- **定制礼物**：刻字首饰/照片T恤/拼图
- **植物**：多肉、小盆栽、鲜花
- **体验类**：按摩券、SPA卡、电影卡

### 送女生的万能选项
- 香薰蜡烛（祖马龙、Diptyque）
- 护手霜礼盒（欧舒丹）
- 口红（选对色号）
- 保温杯（颜值高的）
- 丝巾/发饰

### 送男生的万能选项
- 电动牙刷
- 须后水/香水
- 键盘/鼠标
- 蓝牙音箱
- 零食大礼包

## 避雷指南

### 禁忌清单
- ❌ 钟（送终谐音）
- ❌ 梨（离谐音）
- ❌ 伞（散谐音）
- ❌ 鞋子（送给情侣=送走）
- ❌ 镜子（易碎、招邪）
- ❌ 白色/绿色包装（不吉利）

### 因人而异
- 不知道对方喜好 → 送品质刚需品
- 对方节俭 → 送消耗品（食品、护肤品）
- 对方讲究 → 送品质/小众品牌
