# 礼物参谋

> 送什么好？告诉我，帮你找到贴心又有创意的礼物！

---

## 🎯 这是什么

一个帮你解决"送什么礼物"选择困难症的Skill。

根据收礼人、关系、预算、场合，精准推荐！

---

## 🚀 使用方法

```
"送妈妈什么生日礼物好？"
"情人节送女朋友什么？预算500"
"送老师什么礼物合适？"
"送男性朋友100元左右有什么推荐？"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 收礼人 | 谁收礼物 | 妈妈、男朋友、领导 |
| 关系 | 你们的关系 | 亲密、一般、商务 |
| 场合 | 什么节日/事件 | 生日、情人节、升职 |
| 预算 | 价格范围 | 100以内、500左右 |
| 对方喜好 | （可选）已知偏好 | 爱喝茶、喜欢健身 |

---

## 📦 输出内容

1. **礼物方案** - 3个不同风格推荐
2. **价格说明** - 性价比分析
3. **推荐理由** - 为什么选这个
4. **避雷提示** - 哪些不能送

---

## 💡 适用场景

- 生日不知道送什么
- 节日（情人节/母亲节/父亲节）选礼物
- 送长辈/领导不知道送什么好
- 想送点特别的但没思路

---

## ⚠️ 送礼禁忌

| 禁忌 | 原因 |
|-----|------|
| 钟 | 送终谐音 |
| 梨 | 离谐音 |
| 伞 | 散谐音 |
| 白色包装 | 不吉利 |

---
