# 飞花令

> **来源**: 原创 - 中国古典诗词文化
>
> **灵感来源**: 诗词大会、豆瓣诗词小组、中国传统酒令文化

## 功能

经典中国古诗词飞花令文字游戏。

## 使用方法

```
用户: "飞花令"
助手: 选择令字难度或指定令字开始

用户: "飞花令 春"
助手: 开始春字飞花令
```

## 规则

1. 选定一个关键字（如春、花、月、酒、风等）
2. 轮流说出包含该字的完整诗句
3. 限时回答
4. 答不出者出局

## 示例

```
令字: 月
你: 床前明月光，疑是地上霜 —— 李白
我: 海上生明月，天涯共此时 —— 张九龄
你: 明月松间照，清泉石上流 —— 王维
...
```

## 难度

| 难度 | 范围 |
|-----|------|
| 初级 | 唐诗三百首 |
| 中级 | 唐诗 + 宋词 |
| 高级 | 全唐宋诗词 |
| 地狱 | 含明清诗词 |
