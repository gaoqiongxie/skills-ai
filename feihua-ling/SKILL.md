# 飞花令 Skill

> **来源**: 原创 - 中国古典诗词文化
>
> **灵感来源**: 诗词大会、豆瓣诗词小组、中国传统酒令文化

## 基本信息

- **版本**: 1.0.0
- **作者**: 高琼
- **适用场景**: 诗词学习、文化娱乐、亲子教育、团建活动

## 历史背景

飞花令原本是古代文人雅士在酒宴上行酒令时的一种文字游戏，源自古诗词游戏"曲水流觞"。现代因《中国诗词大会》而广为人知，成为经典的诗词竞技形式。

## 游戏规则

### 基本规则
1. **选定令字**: 行令者选择一个关键字（如"春"、"花"、"月"、"酒"等）
2. **轮流背诵**: 参与者轮流说出包含该字的完整诗句
3. **限时回答**: 通常10秒内需给出答案
4. **说不出罚**: 答不上来者罚酒或出局

### 难度等级
| 难度 | 诗词范围 | 难度说明 |
|-----|---------|---------|
| 初级 | 唐诗三百首 | 常见诗句，耳熟能详 |
| 中级 | 唐诗 + 宋词 | 范围扩大，偶有生僻 |
| 高级 | 全唐宋诗词 | 包括冷门诗句，极具挑战 |
| 地狱 | 扩展至金元明清 | 考验诗词储备量 |

## 工作流程

### 启动游戏
```
用户: "飞花令"
助手: "🌸 飞花令开始！
       
       请选择令字难度：
       1️⃣ 初级 - 常见唐诗（春/花/月/酒）
       2️⃣ 中级 - 唐诗+宋词
       3️⃣ 高级 - 全唐宋诗词
       4️⃣ 地狱 - 包含明清诗词
       
       或者直接说令字，如：'飞花令 春'"
```

### 游戏进行
```
助手: "🌸 飞花令【月】
       
       规则：说出含'月'字的诗句
       每人5秒思考时间
       
       你先来！"
       
用户: "床前明月光，疑是地上霜"
助手: "✅ 正确！李白《静夜思》
       
       下一句："
       
用户: "举头望明月，低头思故乡"
助手: "✅ 正确！李白《静夜思》
       
       ...继续..."
```

### 判定逻辑
1. **检查含字**: 诗句是否包含指定令字
2. **检查格式**: 是否为完整诗句
3. **检查重复**: 是否已背诵过
4. **返回结果**: 正确/错误/提示

### 错误处理
| 错误类型 | 提示示例 |
|---------|---------|
| 不含令字 | "❌ 这句诗里没有'月'字哦～" |
| 诗句错误 | "❌ 这句诗好像不是这样的..." |
| 已背诵过 | "❌ 这句已经说过了，换一句吧" |
