# 朋友圈文案生成器

> 发朋友圈不知道写什么？告诉我场景和风格，帮你生成文案！

---

## 🎯 这是什么

一个帮你生成朋友圈/小红书文案的Skill。

---

## 🚀 使用方法

```
"发朋友圈文案"
"去海边玩，发个朋友圈"
"吃火锅怎么发圈"
"生成小红书文案"
"晒娃文案"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 场景 | 什么情况 | 海边旅行、吃火锅 |
| 风格 | 什么调性 | 文艺、搞笑、简约 |
| 平台 | 朋友圈/小红书 | 小红书 |
| 其他 | 有没有特别要求 | 要带定位 |

---

## 📦 输出内容

1. **文案选项** - 多个版本可选
2. **风格推荐** - 根据场景推荐
3. **Hashtag** - 热门标签
4. **小红书版** - 种草/攻略格式

---

## 💡 功能特点

- 🎨 多种风格可选
- 📱 支持朋友圈/小红书
- #️⃣ 自动推荐Hashtag
- ✨ 小红书爆款公式

---

## 🎤 风格类型

| 风格 | 适合场景 |
|-----|---------|
| 文艺小清新 | 旅行、咖啡、阅读 |
| 搞笑沙雕 | 日常、糗事 |
| 温馨治愈 | 家人、宠物 |
| 精致生活 | 美食、购物 |
| 简约风 | 各种场景通用 |
