# 年终总结生成器

> 回顾这一年，写一份属于你的年度总结！

---

## 🎯 这是什么

一个帮你生成年度总结的Skill。

回顾一年的成就、感恩、遗憾，
展望新的一年。

---

## 🚀 使用方法

```
"帮我写年终总结"
"生成本年度总结"
"这一年怎么总结"
"年度复盘"
"写个年度总结"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 年份 | 哪一年的总结 | 2025 |
| 身份/状态 | 你是什么角色 | 工作第一年 |
| 关键词 | 今年发生的事 | 加薪、升职、考证 |

---

## 📦 输出内容

1. **年度关键词** - 这一年的标签
2. **12个月回顾** - 按月梳理大事
3. **成就清单** - 今年的收获
4. **感恩清单** - 感谢的人
5. **新年愿望** - 下一年的目标

---

## 💡 功能特点

- 📅 按月梳理大事
- 📊 成就数据化
- 💕 温暖有温度
- 🎯 展望新一年

---

## 🎤 总结类型

| 类型 | 说明 |
|-----|------|
| 温馨回忆版 | 按月回忆温暖瞬间 |
| 成就清单版 | 列出所有成就 |
| 数字故事版 | 用数据说话 |

---
