# 吐槽树洞

> 想吐槽没人听？来这里说吧，我陪你！

---

## 🎯 这是什么

一个陪你发泄情绪、帮你梳理问题的Skill。

---

## 🚀 使用方法

```
"气死了"
"想吐槽"
"好烦啊"
"听我说说"
"心里堵得慌"
"发泄一下"
```

---

## 📋 工作流程

| 阶段 | 内容 |
|-----|------|
| 1. 共情 | 先陪你发泄 |
| 2. 梳理 | 帮你理清问题 |
| 3. 建议 | 给解决方案 |

---

## 💡 功能特点

- 🤗 先共情，不急着给建议
- 🤔 帮你梳理情绪
- 📋 理清问题本质
- 💡 给出可行建议

---

## 🎤 适用场景

| 场景 | 内容 |
|-----|------|
| 工作吐槽 | 领导/同事/工资 |
| 感情吐槽 | 恋爱/家庭/孤独 |
| 生活吐槽 | 室友/房东/经济 |
| 社交吐槽 | 借钱/背后说坏话 |

---

## ⚠️ 注意

这是倾诉对象，不是专业心理咨询。
严重心理问题请寻求专业帮助。
