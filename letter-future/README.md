# 给未来写封信

> 写一封给未来自己的信，让时间胶囊封存此刻的心情和期待！

---

## 🎯 这是什么

一个帮你写"给未来自己"信件的Skill。

1年、3年、5年、10年...
写下现在的心情，寄给未来的你。

---

## 🚀 使用方法

```
"写一封给未来自己的信"
"给3年后的自己"
"帮我写封信给1年后的我"
"时间胶囊"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 目标时间 | 1年/3年/5年/10年 | 3年后 |
| 你的状态 | 什么身份/情况 | 上班族/学生 |
| 内容方向 | 有什么想说的 | 想被鼓励 |

---

## 📦 输出内容

1. **完整信件** - 格式规范的信
2. **问题引导** - 帮助你想内容
3. **温暖话语** - 给自己的鼓励
4. **保存建议** - 怎么保存这封信

---

## 💡 功能特点

- ✍️ 格式优美的信件
- 🌱 回顾现在，展望未来
- 💌 温暖的文字，治愈的力量
- 📅 可设定未来日期提醒

---

## 🎤 适用场景

| 场景 | 写给 |
|-----|------|
| 职场新人 | 3年后的自己 |
| 备考学生 | 高考/中考后的自己 |
| 新手父母 | 孩子长大后的自己 |
| 迷茫期 | 走出迷茫后的自己 |
| 任何时刻 | 任何时候的自己 |
