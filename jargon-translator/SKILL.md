## 功能定义

### 核心功能
把各行各业的专业术语翻译成大白话，让你听得懂。

### 输入
- 术语内容（专业词汇/整段话）
- 所属领域（法律/医学/金融/IT/设计等）

### 输出
- 大白话解释
- 通俗类比
- 相关知识

## 使用示例

### 示例1：法律场景

**用户**：律师说的"举证责任倒置"是什么意思？

**AI输出**：
```
📖 【术语翻译】

原术语：举证责任倒置

━━━━━━━━━━━━━━━━━━━━━━━━

大白话解释：

正常情况是"谁告状谁举证"——
你说别人欺负你，你得拿出证据。

"举证责任倒置"就是反过来——
让被告来证明自己没错。

━━━━━━━━━━━━━━━━━━━━━━━━

举个例子：

你去医院治病，结果出了医疗事故。
正常情况下你要证明是医院的错，
但这很难——你不专业、拿不到病历。

举证责任倒置后：
医院必须证明自己没错，
证明不了就是医院的责任。

━━━━━━━━━━━━━━━━━━━━━━━━

为什么这样规定？

因为有些事情，
普通人根本拿不到证据，
所以让"有能力的一方"来举证。

━━━━━━━━━━━━━━━━━━━━━━━━

相关词：
- "举证责任"：谁来提供证据
- "举证期限"：在规定时间内要拿出证据
```

### 示例2：医学场景

**用户**：医生说我"心律不齐"，这是什么意思？

**AI输出**：
```
📖 【术语翻译】

原术语：心律不齐

━━━━━━━━━━━━━━━━━━━━━━━━

大白话解释：

心脏跳动有它自己的节奏，
就像音乐有节拍一样。

"心律不齐"就是：
心脏跳动的节奏乱了，
可能跳得快了、慢了，或者不规律。

━━━━━━━━━━━━━━━━━━━━━━━━

严重吗？

不一定！
有时候是正常的：
- 喝了咖啡/茶
- 紧张/激动
- 熬夜没睡好
- 剧烈运动后

有时候需要重视：
- 经常发作
- 伴有胸闷/头晕
- 老年人

━━━━━━━━━━━━━━━━━━━━━━━━

医生可能会做什么检查：
- 心电图
- 24小时动态心电图
- 心脏彩超

━━━━━━━━━━━━━━━━━━━━━━━━

建议：
- 不要自己吓自己
- 问清楚医生具体情况
- 必要时做进一步检查
```

## 注意事项

### 什么时候需要术语翻译
- 开会听不懂专业词汇
- 看合同看不懂条款
- 读报告不明白数据
- 和专业人士沟通有障碍

### 翻译原则
1. **准确性第一**：确保解释是正确的
2. **通俗易懂**：用生活例子说明
3. **不简化过度**：必要的专业性保留
4. **给出建议**：是否需要进一步了解

### 不能替代
- 正式的法律/医学建议
- 专业咨询
- 重要决策前的专家意见
