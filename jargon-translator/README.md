# 术语翻译器

> 听不懂专业术语？告诉我，我用大白话给你解释！

---

## 🎯 这是什么

一个帮你把专业术语翻译成大白话的工具。

---

## 🚀 使用方法

```
"举证责任是什么意思？"
"心律不齐是什么情况？"
"IT说的API是什么？"
"帮我解释下KPI"
"这是什么术语？"
```

---

## 📋 支持领域

| 领域 | 包含内容 |
|-----|---------|
| 法律 | 合同、劳动法、诉讼 |
| 医学 | 疾病、检查、药物 |
| 金融 | 投资、理财、股市 |
| IT/互联网 | 开发、产品、设计 |
| 商业 | 管理、营销、运营 |
| 设计 | UI/UX、排版、配色 |

---

## 📦 输出内容

1. **大白话解释** - 简单易懂
2. **生活类比** - 用例子说明
3. **是否严重** - 判断是否需要重视
4. **相关建议** - 建议怎么应对

---

## 💡 功能特点

- 🎯 各行各业术语覆盖
- 📖 大白话 + 生活类比
- ⚖️ 保留必要的专业性
- 💡 给出行动建议

---

- 📊 开会听不懂术语
