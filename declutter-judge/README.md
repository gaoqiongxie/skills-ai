# 断舍离裁判

> 纠结这个东西扔不扔？告诉我，我帮你判断！

---

## 🎯 这是什么

一个帮你判断"该扔还是该留"的Skill。

---

## 🚀 使用方法

```
"这件衣服该扔吗？"
"护肤品过期了还能用吗？"
"旅行纪念品要扔吗？"
"帮我整理房间"
"哪些东西要扔？"
```

---

## 📋 判断流程

| 步骤 | 问题 |
|-----|------|
| 1 | 过去1年用过吗？ |
| 2 | 以后会用吗？ |
| 3 | 搬家会带吗？ |

---

## 📦 输出内容

1. **判断结果** - 扔/留/犹豫
2. **判断理由** - 为什么会这样建议
3. **处理方式** - 如果扔，怎么处理
4. **温柔安慰** - 帮你下定决心

---

## 💡 功能特点

- ⚖️ 客观判断标准
- 💭 帮你分析舍不得的原因
- 📦 提供处理建议
- 🌸 温柔引导，不强迫

---

## 🎤 常见判断

| 物品 | 判断 |
|-----|------|
| 5年没穿的衣服 | 扔 |
| 过期的护肤品 | 扔 |
| 旅行纪念品 | 拍照后扔 |
| 旧数据线 | 扔 |
| 纸箱 | 留1-2个就够了 |

---

*扔掉不需要的，留下真正重要的！*
