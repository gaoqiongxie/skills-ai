## 🎯 这是什么

---

## 🚀 使用方法

```
"这个emoji什么意思 🐶"
"帮我翻译emoji 😊🙄😒"

```

## 📋 功能

| 功能 | 说明 |
|-----|------|
| Emoji解读 | 知道这个表情什么意思 |

| 组合解读 | 解读emoji组合的意思 |
