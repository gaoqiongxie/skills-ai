## 功能定义

### 核心功能
把长篇文章/书籍/知识整理成一张张记忆卡片，附带艾宾浩斯遗忘曲线复习提醒。

### 输入
- 知识内容（粘贴文字/描述主题）
- 学习目的（考试/兴趣/工作）
- 目标受众（小学生/初中生/成人）

### 输出
- 知识卡片列表
- 每个卡片的：正面/背面/记忆提示
- 复习计划建议

## 知识卡片格式

```
┌─────────────────────────────────┐
│ 📚 知识卡片                      │
├─────────────────────────────────┤
│                                 │
│ 【正面】                        │
│ 什么是光合作用？                 │
│                                 │
├─────────────────────────────────┤
│                                 │
│ 【背面】                        │
│ 植物利用阳光、二氧化碳和水，     │
│ 制造养料并释放氧气的过程。       │
│                                 │
│ 公式：CO₂ + H₂O + 光 → 有机物 + O₂ │
│                                 │
├─────────────────────────────────┤
│                                 │
│ 【记忆提示】                    │
│ 光 = 阳光                        │
│ 合 = 组合（把各种东西合在一起）   │
│ 作用 = 产生作用的整个过程        │
│ → 想象：植物"喝"阳光长大        │
│                                 │
└─────────────────────────────────┘
```

## 卡片类型

### 1. 问答式（推荐）
- 正面：问题
- 背面：答案
- 适合：考试内容、概念理解

### 2. 概念式
- 正面：概念名称
- 背面：定义+例子+记忆方法
- 适合：术语、专业知识

### 3. 关键词式
- 正面：关键词
- 背面：相关知识点
- 适合：快速复习、串联记忆

### 4. 填空式
- 正面：带空格的句子
- 背面：完整句子
- 适合：记忆细节、公式

## 批量生成功能

### 多篇文章整理

**用户输入**：
```
帮我把以下内容做成记忆卡片：
1. 光合作用是什么
2. 呼吸作用是什么
3. 两者的区别和联系
```

**AI输出**：
- 光合作用卡片 × 3
- 呼吸作用卡片 × 3
- 对比卡片 × 1（区别联系）

## 学习建议

### 高效记忆方法

| 方法 | 说明 |
|-----|------|
| 间隔重复 | 按艾宾浩斯曲线复习 |
| 主动回忆 | 看正面想答案，不要直接看背面 |
| 关联记忆 | 新知识和旧知识联系起来 |
| 讲解给他人 | 能讲出来才是真懂了 |
| 多感官 | 看+读+写+说 |

### 避免的误区

- ❌ 只抄不背
- ❌ 只看不读
- ❌ 一次学太多
- ❌ 从不复习
- ❌ 死记硬背不理解
