# 知识卡片生成器

> 把长文章/书籍整理成一张张记忆卡片，附带复习计划！

---

## 🎯 这是什么

一个帮你把复杂知识整理成"记忆卡片"的Skill。

输入知识，生成卡片，配合艾宾浩斯复习计划，记忆更牢固！

---

## 🚀 使用方法

```
"帮我整理记忆卡片"
"做个知识点背诵卡片"
"整理《登鹳雀楼》的卡片"
"生成复习计划"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 知识内容 | 文字或主题 | 粘贴文章内容 |
| 学习目的 | 考试/兴趣 | 考试 |
| 卡片数量 | 几张 | 5张 |

---

## 📦 输出内容

1. **知识卡片** - 正面问/背面答
2. **记忆提示** - 背诵技巧/口诀
3. **复习计划** - 艾宾浩斯时间表
4. **学习方法** - 高效记忆建议

---

## 💡 功能特点

- 📚 自动提炼核心知识点
- 🎴 卡片格式美观清晰
- 🧠 附带记忆技巧
- 📅 艾宾浩斯复习提醒
- 📝 支持多种学科

---

## ⚡ 适用场景

| 场景 | 效果 |
|-----|------|
| 考试复习 | 重点清晰、快速记忆 |
| 读书记笔记 | 把厚书读薄 |

| 学历史 | 事件+时间+意义 |
