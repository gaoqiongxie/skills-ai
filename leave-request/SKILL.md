## 功能定义

### 核心功能
根据请假原因、时长、领导风格，帮你生成得体又容易批准的请假话术和申请。

### 输入
- 请假类型（年假/病假/事假/婚假/产假/丧假/其他）
- 请假原因（详细或模糊）
- 请假时长（几天）
- 领导风格（可选：严格/开明/难说话）
- 沟通方式（微信/邮件/当面）

### 输出
- 口头话术版本
- 文字申请版本（微信/邮件）
- 注意事项

## 请假类型与话术策略

### 1. 年假

**特点**：正当权益，相对好请

**话术策略**：
```
✅ 直接说：想用年假休息一下
✅ 理由暗示：好久没陪家人了
✅ 工作交代：手头工作已安排好
```

**申请模板**：
```
领导，我想请X天年假（X月X日-X月X日），年假还剩X天。

休假期间：
- [工作安排/交接]
- 紧急联系：[联系方式]

请领导批准，谢谢！
```

### 2. 病假

**特点**：身体原因，必须批

**话术策略**：
```
✅ 实话实说：身体不舒服，医生建议休息
✅ 证明配合：医院证明/病历
✅ 不必多说：不需详细说明什么病
```

**申请模板**：
```
领导，我身体不适（发烧/肠胃炎等），医生建议休息X天。

请假时间：X月X日-X月X日
医院：[医院名称，如有]

工作已交代给[同事姓名]，休假期间手机[畅通/可能接不到，如有重要事找XX]。

请领导批准。
```

### 3. 事假

**特点**：需要正当理由

**话术策略**：
```
✅ 模糊但真诚：家里有点事需要处理
✅ 具体场景：家人来、房子事宜、处理证件等
✅ 强调处理完立刻回来
```

**申请模板**：
```
领导，家里有点私事需要我亲自去处理，想请X天事假（X月X日-X月X日）。

工作已提前安排：
- [交代事项]
- 紧急联系：[同事姓名/联系方式]

请领导批准，事假产生的工资按公司规定扣除即可。
```

### 4. 婚假

**话术策略**：
```
✅ 提前申请：提前1-2周
✅ 祝福接受：感谢领导关心
✅ 工作安排：提前完成或交接
```

**申请模板**：
```
领导，我计划于X月X日（农历X月X）举办婚礼，想请X天婚假。

工作安排：
- 婚礼前完成XXX工作
- XXX工作由[同事]暂时接管

感谢领导一直以来对我的照顾，婚礼邀请您有时间来参加~

请领导批准。
```

### 5. 产假/陪产假

**话术策略**：
```
✅ 提前申请
✅ 证明材料齐全
✅ 工作交接详细
```

### 6. 丧假

**话术策略**：
```
✅ 简洁直接：家里有亲人过世
✅ 不需详细说明关系
✅ 尽快赶回去
```

**申请模板**：
```
领导，家中[爷爷/奶奶/外婆/外公]去世，需回家处理后事，想请X天丧假。

[如已出发可简短]：已出发，预计X月X日返岗。
[如未出发]：工作已交接给[同事]。

请领导批准。
```

## 沟通场景话术

### 当面沟通（正式）

**开场**：
```
领导，有个事想跟您说一下，方便吗？
```

**正文**：
```
是这样的，我[请假原因]，想请X天假。
时间是X月X日到X月X日。
工作我已经安排好了，[具体交代事项]。
您看可以吗？
```

**结尾**：
```
好的，谢谢领导理解！
```

### 微信/钉钉沟通（简洁）

```
[表情]领导，想跟您请X天假（X月X日-X月X日），原因是[简短说明]。

工作已安排好，紧急事情可以联系[同事/电话]。

谢谢领导！
```

### 邮件沟通（正式）

```
邮件主题：【请假申请】XXX X月X日-X月X日请假

尊敬的XX领导：

您好！因[原因]，拟于X月X日至X月X日请假X天。

请假期间，工作已妥善安排：
1. XXX工作由[同事]跟进
2. XXX事项已提前完成
3. 紧急联系：[联系方式]

请领导批准！

此致
敬礼

申请人：XXX
X月X日
```

## 请假理由库

### ✅ 好用的理由
| 理由 | 适用场景 | 可信度 |
|-----|---------|-------|
| 身体不舒服 | 任何时候 | ⭐⭐⭐⭐⭐ |
| 家里有事 | 事假 | ⭐⭐⭐⭐ |
| 家人来/探亲 | 探亲 | ⭐⭐⭐⭐ |
| 房子过户/装修 | 买房/卖房 | ⭐⭐⭐⭐ |
| 处理证件（身份证/驾照等） | 官方事务 | ⭐⭐⭐⭐ |
| 孩子学校活动 | 有孩子 | ⭐⭐⭐ |
| 预约的重要事情 | 牙医/体检等 | ⭐⭐⭐ |

### ⚠️ 慎用的理由
| 理由 | 风险 |
|-----|------|
| "朋友结婚" | 可能被问东问西 |
| "心情不好想休息" | 可能被认为不专业 |
| "出去旅游" | 可能被认为不重视工作 |

## 注意事项

### ✅ 应该做的
1. **提前说**：尽量提前1-3天请假
2. **工作到位**：提前交代/完成工作
3. **态度诚恳**：不要像休年假是应该的
4. **渠道正确**：按公司规定渠道请假
5. **留好证据**：截图保存审批记录

### ❌ 不应该做的
1. 临时翘班不请假
2. 假后补请假（很尴尬）
3. 请假理由前后矛盾
4. 请假期间完全失联（紧急联系要保持）
5. 假后第一天就抱怨（领导印象会差）

## 不同领导风格应对

| 领导风格 | 应对策略 |
|---------|---------|
| 严格型 | 提前申请、理由充分、工作到位 |
| 开明型 | 直接说、正常安排工作即可 |
| 难说话型 | 强调工作已安排、表示愿意配合加班 |
| 完美主义型 | 给他看你的交接清单，打消顾虑 |
