# 请假话术助手

> 请假不知道怎么开口？告诉我原因和时长，帮你生成得体话术！

---

## 🎯 这是什么

一个帮你生成得体、容易获批请假话术的Skill。

---

## 🚀 使用方法

```
"帮我写请假申请"
"怎么跟领导请假？"
"年假怎么请？"
"病假怎么说？"
"请事假什么理由好？"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 请假类型 | 年假/病假/事假/婚假 | 年假 |
| 请假原因 | 详细或模糊 | 家里有事 |
| 请假时长 | 几天 | 3天 |
| 沟通方式 | 微信/邮件/当面 | 微信 |
| 领导风格 | （可选） | 严格/开明 |

---

## 📦 输出内容

1. **口头话术** - 当面怎么说
2. **文字申请** - 微信/钉钉/邮件版
3. **注意事项** - 哪些雷区别踩
4. **工作安排** - 怎么交代工作

---

## 💡 功能特点

- 📝 支持多种请假类型
- 💬 提供口头/文字双版本
- 👔 根据领导风格调整话术
- ⚠️ 提醒注意事项
