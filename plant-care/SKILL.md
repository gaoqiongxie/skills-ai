## 功能定义

### 核心功能
提供常见绿植的养护指南，帮你诊断植物问题并给出解决方案。

### 输入
- 植物名称
- 植物出现的问题（症状）
- 养护环境描述

### 输出
- 养护指南
- 问题诊断
- 解决方案

## 常见植物养护

### 1. 绿萝

```
🌿 【绿萝养护指南】

📍 基本信息
- 难度：⭐（几乎养不死）
- 喜阴：⭐⭐⭐⭐⭐
- 浇水：见干见湿

☀️ 光照
- 喜阴，避免阳光直射
- 散射光最佳
- 放室内任何位置都行

💧 浇水
- 土干了再浇
- 夏天1周1-2次
- 冬天2周1次
- 盆底不能积水

🌡️ 温度
- 15-30°C最佳
- 不要低于5°C
- 北方冬天注意保暖

🪴 土壤
- 疏松透气
- 通用营养土即可

💩 施肥
- 生长期（春夏）每月1次
- 稀释的液肥
- 冬天不施肥

✂️ 修剪
- 枯叶及时剪掉
- 藤太长可以剪下来扦插
```

### 2. 多肉

```
🌵 【多肉养护指南】

📍 基本信息
- 难度：⭐⭐⭐
- 喜阳：⭐⭐⭐⭐⭐
- 浇水：宁干勿湿

☀️ 光照
- 喜阳，每天至少4小时光照
- 夏天避免暴晒（遮阴）
- 春秋可全日照

💧 浇水
- 宁干勿湿！
- 叶片软了再浇
- 夏天：1-2周1次
- 冬天：1月1次
- 浇到盆底出水就行

🌡️ 温度
- 5-30°C
- 夏天超过35°C要遮阴
- 冬天低于5°C要搬室内

🪴 土壤
- 透气透水
- 颗粒土+泥炭土（1:1）

💩 施肥
- 生长期（春秋）每月1次
- 稀释的液肥/缓释肥
- 夏天冬天不施肥

⚠️ 常见问题
- 徒长：光照不够
- 化水：浇水太多
- 黑腐：浇水太多+不通风
```

### 3. 君子兰

```
🌸 【君子兰养护指南】

📍 基本信息
- 难度：⭐⭐⭐⭐
- 喜散光：⭐⭐⭐⭐
- 浇水：适中

☀️ 光照
- 喜散射光，怕暴晒
- 放东/北窗台最佳
- 夏天遮阴

💧 浇水
- 土壤半干再浇
- 夏天：1周1次
- 冬天：2-3周1次
- 沿着盆边浇，不要浇到叶心

🌡️ 温度
- 15-25°C最佳
- 冬天需要低温春化（5-10°C）
- 夏天要通风

🪴 土壤
- 疏松透气
- 腐叶土+松针+河沙

💩 施肥
- 生长期（春秋）每月1次
- 磷钾肥为主（促进开花）

✂️ 修剪
- 及时清除枯叶
- 花箭剪掉（留种除外）
```

### 4. 吊兰

```
🌱 【吊兰养护指南】

📍 基本信息

🌡️ 温度

💩 施肥

✂️ 繁殖
```

### 5. 龟背竹

```
🌿 【龟背竹养护指南】

- 喜阴：⭐⭐⭐⭐⭐
```

## 植物问题诊断

### 1. 叶片发黄

```
🔍 【诊断】叶片发黄

可能原因：

① 浇水过多
症状：老叶先黄，新叶软
解决：控水，加强通风

② 浇水过少
症状：叶尖发黄，土壤干燥
解决：及时浇水，浇透

③ 光照问题
症状：暴晒 → 焦黄
      缺光 → 新叶发黄
解决：调整位置

④ 缺肥
症状：整体黄叶，脉绿叶黄
解决：补充肥料

⑤ 正常代谢
症状：底部老叶发黄
解决：正常现象，摘掉即可
```

### 2. 叶片蔫了

```
🔍 【诊断】叶片蔫了

可能原因：

① 缺水
症状：土壤干，叶片卷曲
解决：浇水，浇透

② 水多烂根
症状：土壤湿，叶片软
解决：控水，检查根部

③ 高温
症状：天热时蔫
解决：移到阴凉处

④ 冻伤
症状：低温后蔫
解决：保暖，避免冷风
```

### 3. 叶片有斑点

```
🔍 【诊断】叶片有斑点

可能原因：

① 晒斑
症状：焦褐色斑点
解决：避免阳光直射

② 病虫害
症状：小虫/蜘蛛网/白粉
解决：喷药治疗

③ 浇水问题
症状：叶尖/叶缘褐色
解决：调整浇水方法

④ 施肥过多
症状：叶面灼伤
解决：多浇水稀释
```

### 4. 不生长

```
🔍 【诊断】植物不生长

可能原因：

① 休眠期
症状：秋冬季不生长
解决：正常现象

② 根系问题
症状：不长新叶
解决：检查根部，修根换土

③ 缺光
症状：徒长但不长
解决：增加光照

④ 缺肥
症状：生长缓慢
解决：补充肥料

⑤ 盆太小
症状：很久没换盆
解决：换大盆
```

## 四季养护要点

### 春季（3-5月）

```
🌸 【春季养护】

- 换盆换土的好时机
- 逐渐增加浇水
- 开始施肥
- 增加光照
- 繁殖的好季节（扦插/分株）
```

### 夏季（6-8月）

```
☀️ 【夏季养护】

- 注意遮阴（避免暴晒）
- 增加浇水频率
- 多通风
- 适当喷水降温
- 减少施肥（高温休眠）
- 病虫害高发期，注意防治
```

### 秋季（9-11月）

```
🍂 【秋季养护】

- 逐渐减少浇水
- 增加光照
- 适当施肥
- 适合换盆
- 繁殖的好季节
- 准备过冬
```

### 冬季（12-2月）

```
❄️ 【冬季养护】

- 减少浇水
- 停止施肥
- 注意保暖
- 多晒太阳
- 避免冷风
- 北方注意室内湿度
```

## 使用示例

### 示例

**用户**：我的绿萝叶子发黄了，怎么回事？

**AI输出**：
```
🌿 【植物诊断】绿萝叶子发黄

━━━━━━━━━━━━━━━━━━━━━━━━

🔍 可能原因排查：

请对照以下症状：

① 【浇水过多】⚠️ 最常见
表现：
- 老叶从叶尖开始发黄
- 新叶颜色淡但不是马上黄
- 土壤一直湿湿的
- 茎可能有点软

→ 怎么判断：翻开土看看，是不是一直很湿？

② 【浇水过少】
表现：
- 叶尖发黄、干枯
- 土壤干燥发白
- 叶片卷曲

③ 【缺光】
表现：
- 新长出的叶子发黄
- 茎细长徒长

④ 【正常代谢】
表现：
- 底部的叶子发黄
- 只有1-2片

⑤ 【缺肥】
表现：
- 整株发黄
- 叶脉绿色但叶肉发黄

━━━━━━━━━━━━━━━━━━━━━━━━

💡 请问：
1. 土壤是湿的还是干的？
2. 发黄的是新叶还是老叶？
3. 发黄是什么样的？（整片黄/叶尖黄/斑点黄）

━━━━━━━━━━━━━━━━━━━━━━━━

📋 通用建议：

✓ 绿萝浇水原则：
  手指插进土里，干了再浇
  浇到盆底出水就行

✓ 绿萝喜阴：
  放室内任何位置都行
  但要有散射光

✓ 施肥：
  生长期（春夏）每月1次
  稀释的液肥即可
```

## 养植物基本原则

```
📋 【黄金法则】

1. 🌡️ 了解植物习性
   - 喜阴还是喜阳？
   - 喜湿还是耐旱？

2. 💧 浇水原则
   - 大多数植物：宁干勿湿
   - 手指插土法：干了再浇
   - 浇则浇透

3. ☀️ 光照
   - 耐阴 ≠ 完全不需要光
   - 散射光 = 室内明亮处

4. 🪴 通风
   - 通风很重要！
   - 避免闷热潮湿

5. 💩 施肥
   - 薄肥勤施
   - 生长期多施，休眠期不施

6. 🐛 病虫害
   - 早发现早处理
   - 隔离病株
```
