# 绿植养护指南

> 养什么死什么？告诉我你的植物怎么了，我来帮你诊断！

---

## 🎯 这是什么

一个帮你养好绿植的Skill。

植物生病了？不知道怎么养？
告诉我症状，我来帮你！

---

## 🚀 使用方法

```
"绿萝怎么养"
"多肉怎么浇水"
"植物叶子发黄"
"植物蔫了怎么办"
"帮我看看这盆花"
```

---

## 📋 输入要素

| 要素 | 说明 | 示例 |
|-----|------|------|
| 植物名 | 什么植物 | 绿萝 |
| 问题症状 | 怎么了 | 叶子发黄 |
| 环境描述 | 放在哪 | 北阳台 |

---

## 📦 输出内容

1. **养护指南** - 浇水/光照/温度/施肥
2. **问题诊断** - 叶子发黄/蔫了/不长
3. **解决方案** - 怎么救
4. **四季要点** - 春/夏/秋/冬

---

## 💡 功能特点

- 🌱 常见植物养护指南
- 🔍 植物问题诊断
- 💧 浇水指导
- ☀️ 光照建议
