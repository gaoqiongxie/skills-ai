# 🪂 技能加速器 · 滑翔伞学习法

> 来源：一次真实的滑翔伞训练体验
> 核心：安全失败 · 快速迭代 · 真实验证

---

## 💡 方法论来源

2026年4月周末，一次滑翔伞训练的完整流程：

```
理论 → 实践 → 复盘 → 模拟器 → 复盘 → 放飞
```

这不是随机排列的6个步骤，而是一套经过反复验证的**技能习得方法论**。把它提炼出来，适用于任何技能的学习。

---

## 🎯 核心价值

| 问题 | 本方法解决方式 |
|------|-------------|
| 学了很久没进步 | 强制"放飞"验证，不让自己停留在理论 |
| 练习没有针对性 | 复盘→找弱点→模拟器专项攻克 |
| 真实场景压力大 | 模拟器积累"安全失败"的经验 |
| 不知道从哪里开始 | 60分起步，边飞边调整 |

---

## ⏰ 六阶段总览

| 阶段 | 关键词 | 时间占比 |
|------|--------|---------|
| 1. 理论 | 建地图 | 15% |
| 2. 实践 | 碰真实 | 20% |
| 3. 复盘① | 找差距 | 5% |
| 4. 模拟器 | 刻意练 | 40% ← 核心 |
| 5. 复盘② | 固化它 | 5% |
| 6. 放飞 | 真验证 | 15% |
