## 游戏规则

### 基本规则
1. **首字对接**: 后一个成语的首字 = 前一个成语的尾字
2. **同音可接**: 首字与尾字同音时也可接受
3. **禁止重复**: 已说过的成语不能再说
4. **超时判负**: 10秒内说不出成语判负

### 示例
```
玩家A：画蛇添足
玩家B：足智多谋  ✅ (足=足)
玩家A：谋财害命  ✅ (命≠足，同音谋)
玩家B：名落孙山  ✅ (山=山)
...
```

### 变体规则
| 变体 | 规则说明 |
|-----|---------|
| 谐音接龙 | 允许首尾字同音不同字 |
| 典故接龙 | 必须说出成语背后的典故 |
| 接龙升级 | 从二字成语到四字成语递进 |
| 限时接龙 | 每轮限时回答 |

## 工作流程

### 启动游戏
```
用户: "成语接龙"
助手: "🎮 成语接龙开始！
       
       规则：
       1. 我说一个成语
       2. 你接下一个，首字=我的尾字
       3. 同音可接，禁止重复
       
       开始：
       【安居乐业】
       
       请接龙！"
```

### 判定逻辑
1. **检查格式**: 是否为四字成语
2. **检查首字**: 是否与上成语尾字匹配
3. **检查重复**: 是否已用过
4. **返回结果**: 通过/失败/建议

### 错误处理
| 错误类型 | 提示示例 |
|---------|---------|
| 首字不匹配 | "❌ 'XXX'的首字是'X'，需要接'X'开头的成语" |
| 非成语 | "❌ 'XXX'不是成语，请换一个" |
| 已重复 | "❌ 这个成语用过了，请换一个" |

## 注意事项

1. **优先常用成语**: 选择常见、易理解的成语
2. **注意多音字**: 如"行"可读xing/hang，"得"可读de/děi
3. **保护玩家体验**: 适当放水或提示，增加趣味性
4. **记录战绩**: 可以统计连胜次数
